trpFill

브라우져 크기로 화면을 채움.

trpFill([options])

설명
trpFill.js 불러오면 trpFill, 를 사용할수 있습니다.
<script type="text/javascript" src="./js/libs/trpFill.js"></script>
가로세로중 작은 크기에 비례하여 채워집니다.
windows resize 이벤트에 반응합니다.
class 설명
options 이름 options 설명
targetWidth 타겟 가로 크기 (필수)
targetHeight 타겟 세로 크기 (필수)
triggerTarget width, height가 참조되서 채워질 jQuery선택자(#id, .class) (기본값은 브라우저객체 크기)
