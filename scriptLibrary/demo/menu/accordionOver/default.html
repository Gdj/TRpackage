accordionOver menu

over된 메뉴만 활성화되는 아코디언 메뉴

.trpAccordionOverMenu(overarea:string , contant:string, activeClass:string )

설명
기본 위치값은 CSS 에서 설정해야합니다.
imAccordionOver menu 설명
매개변수 매개변수 설명
overarea 아코디언 메뉴 오버됬을때 영역 선택자
contant 아코디언 메뉴의 컨텐츠 선택자
activeClass 메뉴의 활성화 클래스 이름
기본값은 "on"클래스로 되여있습니다.
tab menu 설명
메서드 메서드 설명
setBtn(index) 인덱스번호로 아코디언 메뉴를 활성화 합니다.
