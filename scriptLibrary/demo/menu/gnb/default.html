GNB

GNB 네비게이션 : 서브 뎁스가 우측에 세로로 정열됩니다.

.trpGNB( $activeClass:string, $activeFn:function )

설명
기본 위치값은 CSS 에서 설정해야합니다.
quick menu 설명
매개변수 매개변수 설명
$activeClass 메뉴의 활성화 클래스 이름
기본값은 "on"클래스로 되여있습니다.
$activeFn($index,event) 1뎁스 리턴함수, 마우스 이벤트
