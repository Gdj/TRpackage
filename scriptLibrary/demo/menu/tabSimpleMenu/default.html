tab menu

tab 버튼을 클릭하면 해당내용의 컨텐츠가 보이는 형태의 메뉴

.trpTabSimpleMenu( button, contant [, activeClass] )

설명
기본 위치값은 CSS 에서 설정해야합니다.
ul li 구조의 텝메뉴 형태여야 합니다.
tab menu 설명
매개변수 매개변수 설명
button 텝메뉴의 버튼 선택자
contant 텝메뉴의 컨텐츠 선택자
activeClass 메뉴의 활성화 클래스 이름
(기본값 "on" 클래스)
tab menu 설명
메서드 메서드 설명
setBtn(index | array) 인덱스번호로 텝메뉴를 활성화 합니다.
중첩텝의 경우 배열 형태로 활성화 값을 줄수있습니다.
