trpDropdown

드롭다운 메뉴

trpDropdown([options])

설명
trpFrom.js 불러오면 trpDropdown, 를 사용할수 있습니다.
<script type="text/javascript" src="./js/libs/trpFrom.js"></script>
class 설명
options 이름 options 설명
list 보여질 list 영역 선택자 default(".trp-dropdown_list-box")
area dropdown 컨테이너 영역 선택자 default(".trp-dropdown-area")
class 설명
method 이름 method 설명
getOpen() 드롭다운 컨텐츠 보임
getClose 드롭다운 컨텐츠 숨김

trpJsDropdownManager

설명
trpFrom.js 불러오면 trpDropdown, 를 사용할수 있습니다.
<script type="text/javascript" src="./js/libs/trpFrom.js"></script>
class 설명
method 이름 method 설명
trpJsDropdown($id) trpJsDropdownManager.dropdown($id) 과 동일한기능
.dropdown($id) 드롭다운 등록
.init() 드롭다운 이벤트 초기화
.destroy() 트롭다운 이벤트 삭제
.closeAll() 모든드롭다운 닫기
.onSelect($callbackFn($selectObj)) 트롭다운 선택 아이템 콜백 함수()
$selectObj = { id : $id, text : $text, value : $value }
