trpLimitText

텍스트 바이트를 초과시 함수 리턴함.

trpLimitText( {options} )

설명
최대 바이트, 현재 바이트수 표시선택자, 최대 바이트수 표시선택자, 최대바이트 초과시 호출 함수를 호출
class 설명
options 이름 options 설명
maxByte 최대 수용할 수 있는 바이트
currentEle 현재 바이트 표시될 테그선택자.
maxByte 최대 바이트 표시될 테그선택자.
onMaxReturn($byte) 최대 수용할 수 있는 바이트를 초과시 호출될 함수. (초과된 바이트 리턴)
