timeCounter

아래 두 플러그인으로 timeCount 한다.
trpCounter 총값 안에서 루프 카운트를 한다.
trpTimeProgress 시간을 백분율로 바꾸어준다.

trpCounter( total, $callFn )

설명
참고값에 변경하고자 하는 element를 선택해 사용.
<script type="text/javascript" src="./js/libs/trpUnit.js"></script>
param 설명
param 이름 param 설명
$total 반복될 겟수
$callFn( index) 이벤트가 일어나면 호출되는 콜백함수.
index : 현재 인덱스 값
method 설명
method 이름 method 설명
setPrev 이전 인덱스
setNext 다음 인덱스
setIndex($index) 인덱스 변경
getTC().total 현재 총겟수
getTC().index 현재 인덱스

trpTimeProgress( {options} , fractionCall($per) )

설명
참고값에 변경하고자 하는 element를 선택해 사용.
<script type="text/javascript" src="./js/libs/trpUnit.js"></script>
param 설명
param 이름 param 설명
$options.interval 퍼센트로 변화될시간, (5000)기본값 == 5초
$options.call 갱신 시간, (100)기본값.
$fractionCall(per) 콜백함수로 백분율 리턴.
method 설명
method 이름 method 설명
play 이전 인덱스
pause 다음 인덱스
stop 인덱스 변경
Event 설명
Event 이름 Event 설명
event_trpTimer_end 100%, 혹 넘어가면 호출됩니다.
