layer fixed Popup

layer Popup 버튼을 클릭하면 layer Popup 열림

.trpLayerFixedPopupFile( $popupUrl, $tarPopup )

설명
기본 위치값은 CSS 에서 설정해야합니다.
options 설명
param 이름 param 설명
$popupUrl 팝업 URL및 선택자
$tarPopup 팝업 선택자입니다.
값이 없을때는 "href" 앵커 값으로 선택됩니다.
layer Popup 설명
메서드 메서드 설명
open($this) 팝업을 보이게 합니다
$this : 클릭된 버튼을 넘깁니다. (팝업이 닫치게 되면 넘긴 값으로 포커스가 이동합니다.)
close() 팝업을 숨김니다.
getBtn 팝업 버튼 선택자
