masonry

masonry 이용하여 유동적인 레이아웃을 만들수 있다.
link : masonry jQuery Plugin(GitHub)

.masonry( { option } )

설명
기본 위치값은 CSS 에서 설정해야합니다.
버튼 옵션 정의
masonry 옵션
매개변수 매개변수 설명
columnWidth 수평 그리드의 컬럼의 폭(수치, 클래스명)
itemSelector 아이템 요소로 사용하도록 지정되는 자식 요소
gutter 항목 요소 사이의 수평 공간(수치, 클래스명)
isFitWidth 기본값은 false 이며 true 로설정하면 CSS로 컨테이너 위치를 지정 할 수 있다.
(아이템의 부모 컨테이너에 { margin : 0 auto; } 중앙 정열)
isInitLayout 기본값은 true 이며 false 로설하면 초기에 레이웃을 사용하지 않도록합니다.
isOriginLeft 기본값은 true 이며 false 로설정하면 오른쪽기준으로 정열 됩니다.
isOriginTop 기본값은 true 이며 false 로설정하면 아래쪽기준으로 정열 됩니다.
stamp 지정요소를 피해서 배치하도록 합니다.
transitionDuration 기본 값은 '0.4s' 이며 아이템 들의 움직이는 시간을 변경할수있다.
masonry 메서드
메서드 메서드 설명
masonry 정열합니다.
addItems 아이템이 뒤에 추가됍니다.
appended 아이템이 뒤에 추가되고 정열됩니다.
prepended 아이템이 앞에 추가되고 정열됩니다.
stamp 스템프 설정
스템프를 피해서 정열됩니다.
unstamp 스템프 해제
스템프를 무시하고 정열합니다.
unbindResize 리싸이즈에 반응 안하도록합니다.
bindResize 리싸이즈에 반응 하도록합니다.
masonry 이벤트
이벤트 이벤트 설명
layoutComplete 레이아웃 위치 전황이 완료 됐을때 호출
removeComplete 아이템이 제거되었을때 호출

function getItemElement() {
  var elem = document.createElement('div');
  var wRand = Math.random();
  var hRand = Math.random();
  var widthClass = wRand > 0.92 ? 'w4' : wRand > 0.8 ? 'w3' : wRand > 0.6 ? 'w2' : '';
  var heightClass = hRand > 0.85 ? 'h4' : hRand > 0.6 ? 'h3' : hRand > 0.35 ? 'h2' : '';
  elem.className = 'item ' + widthClass + ' ' + heightClass;
  return elem;
}
