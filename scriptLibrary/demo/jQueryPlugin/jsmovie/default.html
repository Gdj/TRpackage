jsMovie

이미지 시컨스를 연속재생 시켜줍니다.
link API : http://jsmovie.konsultaner.de/

.jsMovie({ option } )

설명
이미지로된 숫자를 호전하듯이 정열된는 효과.
관련 css가 선언되어야 합니다.
option
option 이름 option 설명
images : [] 배열로 이미지 이름을 넣을 수 있습니다. 단, 이름에 공백이 들어가서는 안됩니다.
sequence : '' 이미지 시컨스를 자동으로 배열로 만들어 줍니다.
image####.jpg("#"의 겟수는 카운터 자리수만큼 넣어줍니다.)
from:0 시작하는 시컨스 번호.
to:1 끝나는 시컨스 번호.
step:1 시컨스의 스텝을 정수로 정의합니다.
folder:"pic/" 이미지 홀더 경로를 문자열로 정의합니다.
grid: 연속이미지 지도의 가이드
{ height:800, width:600, rows:1, columns:1 }
showPreLoader:false true 이면 프리로더 표시
loader: 로딩 이미지의 가이드정보
{ path:"이이지경로", width:연속이미지1개 width, height:연속이미지1개 height, columns:연속이미지 가로겟수 , row:연속이미지 세로겟수 }
fps:12 초당 이미지 겟수
width:640 연속 이미지 width 크기
height:480 연속 이미지 height 크기
loadParallel:1 평행 이미지 양 정수값?
repeat:true 반복여부 boolean 값
playOnLoad:false true 이면 이미지로드되면 자동 플레이된다.
performStop:true true 이면 첫프레임으로 돌아가지않고 일시 정지상태로 있는다.
playBackwards:false true 이면 거꾸로 재생된다.
verbose:true true 이면 자세한 이밴트를 가져 옵니다.
verbose:[] 초기화에 전달되는 목록
메서드
메서드 이름 메서드 설명 : $("시컨스컨테이너 선택자").jsMovie("메서드 이름")
realFps 실제 프라우저의 fps를 적용.
play 재생 합니다.
$(".movie").jsMovie("play",(from),(to), (repeat), (performStop) );.
pause 재생을 일시 정지합니다.
stop 재생을 정지 합니다.
gotoFrame 특정 프레임으로 이동합니다.
$(".movie").jsMovie("gotoFrame",(프래임 번호));
nextFrame 다음 프레임 이동.
previousFrame 이전 프레임 이동.
addClip 특정위치를 만듭니다.
$(".movie").jsMovie("addClip","클립이름",(from),(to), (지연시간 1/1000) );
playClip 클립 위치에서 실행합니다.
$(".movie").jsMovie("playClip","클립이름");
$(".movie").jsMovie("playClip",function(){return {start:10,end:20,pause:1000} });
$(".movie").jsMovie("playClip",(클립 인덱스));
playClips 모든 클립재생.
getClip 클립 이름으로 글립객체를 가져옵니다.
removeClip 특정글립을 삭제 합니다.
getClipQueue 모든 클립오브젝트 반환
destroy jsMovie를 제거하여 초기화함.
EVENT
EVENT 이름 EVENT 설명
play 재생이 시작하면 호출 됩니다.
pause 일시정지 하면 호출 됩니다.
stop 재생 정지 되면 호출 됩니다.
ended 마지막 프레임에 도달되면 호출 됩니다.
playing 재생 중일때 호출 됩니다.
loaded 이미지가 모두 로드 되면 호출 됩니다.
verbose, function(e,output) 모든 이벤트 및 텍스트를 포함하여 호출 됩니다.

 <head>

		.controler img { width:50px; }

  <script type="text/javascript" src="../../../../asset/js/libs/jquery-1.9.1.min.js" ></script>
  <script type="text/javascript" src="../../../../asset/js/libs/jsmovie/jquery.jsmovie.1.4.4.js"></script>
  <script type="text/javascript">
	$(document).ready(function(){


		var mmm = $('#movie').jsMovie({
		sequence: 'scene_#####.jpg',	// 시퀀스 형식
		from: 1,						// 시퀀스 첫 프레임
		to: 71,							// 시퀀스 마지막 프레임
		folder : "./images/sequence/", 	// 시퀀스 이미지경로
		width:(640),				// 플레이어 세로크기
		height:(282),				// 플레이어 세로크기
		fps:30,						// 프레임 속도
        playOnLoad : false,         // 로드후 자동 플레이 하지 않음
		showPreLoader : true,       // 로더 보이게 
		loader: {path:"../../../../asset/js/libs/jsmovie/loader.png", height:40, width:40, rows:2, columns:4}, // 로딩 이미지 세팅
		repeat:false				// 반복하지 않음
		});
	
		/* 컨트롤 */
		$(".btn_play").on('click',function(){ $('#movie').jsMovie('play') });
		$(".btn_pause").on('click',function(){ $('#movie').jsMovie('pause'); });
		$(".btn_stop").on('click',function(){ $('#movie').jsMovie('stop'); });
	
		
		/* 이벤트 */
		$('#movie').on('ended', function(v) { 
			console.log("플레이 끝!");
			$('#movie').jsMovie('pause');
			$("#movie").jsMovie("gotoFrame",71); 
		});
		$('#movie').on('loaded', function() { 
			console.log("로드 완료!");			
		});
			
							
	});// document.ready
  </script>
