trpStickyTopEvent, trpStickyBottomEvent

스크롤 움직임에 따라 텝메뉴 활성화.

trpStickyTopEvent( $stickyPoint , $parentClass ), trpStickyBottomEvent( $stickyPoint , $parentClass )

설명
trpLayout.js 불러오면 trpTabScrollMenu 사용할수 있습니다.
<script type="text/javascript" src="./js/libs/trpLayout.js"></script>
parameter 설명
parameter 이름 parameter 설명
$stickyPoint trpStickyTopEvent 일시 top 값,
trpStickyBottomEvent 일시 bottom 값
$parentClass 부모 클래스에 스티키 클래스를 추가한다.
return method 설명
method 이름 method 설명
setPoint($stickyPoint) stickyPoint 값을 변경할 수 있다.
trpStickyTopEvent 설명
Event 이름 Event 설명 $stickyElementBottom.on('stickyOn', function(event, scrollPosition) {});
stickyOn(event, scrollPosition) 스티키 되는 순간 발생
stickyOnIng(event, scrollPosition) 스티키 진행 되는 중.. 발생
stickyUnder(event, scrollPosition) 스티키 벗어 나는 순간 발생
