trpScrollPositionFn

타켓 위치에 도달하면 true 그렇지 않으면 false 파라미터 값을 넘긴다.

$("body").trpScrollPositionFn( $scrollTar, $scrollTarModi, $functionChange);

설명
타겟 위치에서 윈도우 높이 절반 기준으로 클래스 추가, 못미치면 삭제
trpLayout.js 불러오면 trpScrollPositionFn 사용할수 있습니다.
<script type="text/javascript" src="./js/libs/trpLayout.js"></script>
class 설명
파라미터 이름 파마미너 설명
$scrollTar 기준 타겟 위치 선택자
$scrollTarModi 기준 타겟 위치 가감 수치
$functionChange($b) 실행함수 $function( over:true | under:false )
