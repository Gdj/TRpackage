trpScrollSyncTop

스크롤 움직임에 따라 타겟 높이 값을 변경하여 따라다니게 한다. (상단기준으로 계산)

trpScrollSyncTop( option )

설명
trpLayout.js 불러오면 trpScrollSync 사용할수 있습니다.
<script type="text/javascript" src="./js/libs/trpLayout.js"></script>
class 설명
option 이름 option 설명
positionTar 적용할 컨텐츠 변경시 (기본값 : this)
widthTar 적용할 타켓 넓이값 (기본값 : this.width)
heightTar 적용할 타켓 높이값 (기본값 : this.height)
topStart 고정이 시작될 위치 (기본값 : $(window).height())
topMagin 고정이 시작될 위치 + @간격 (기본값 : 0)
bottomStop 적용할 타겟 하단기준 멈춰야할 위치 (기본값 : 300)
method 설명
method 이름 method 설명
setWidthTar 적용할 타켓 넓이값 변경
setHeightTar 적용할 타켓 높이값 변경
setTopStart 고정이 시작될 위치 변경
settopMagin 고정이 시작될 위치 + @간격 변경
setBottomStop 적용할 타겟 하단기준 멈춰야할 위치 변경
getPosition 타겟 위치 확인 getPosition.top
이벤트 클래스
이벤트 class class 설명
hold_start 고정이 시작할때
hold_stop 고정이 끝나 지정위치 도달했을때
