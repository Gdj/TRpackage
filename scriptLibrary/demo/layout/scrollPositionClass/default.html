trpScrollPositionClass

타켓 위치에 도달하면 특정 엘리먼트에 클래스를 추가, 못미치면 삭제한다.

$("body").trpScrollPositionClass( $scrollTar, $scrollTarModi, $addTar , $addClass);

설명
trpLayout.js 불러오면 trpScrollPositionClass 사용할수 있습니다.
<script type="text/javascript" src="./js/libs/trpLayout.js"></script>
class 설명
파라미터 이름 파마미너 설명
$scrollTar 기준 타겟 위치 선택자
$show_per 컨테이너 크기대비 보여질 위치 (.5)
$addTarClass 추가될 클래스 이름
$instantB false : 페이지 하단 밖으로 밀려날 때 초기화
true : 시작 시점 하단 일때 초기화
class 설명
method 이름 method 설명
setTarModi 타겟 위치 가감 (px) 위치
