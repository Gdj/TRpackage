trpTabScrollMenu

스크롤 움직임에 따라 텝메뉴 활성화.

trpTabScrollMenu( parameter )

설명
trpLayout.js 불러오면 trpTabScrollMenu 사용할수 있습니다.
<script type="text/javascript" src="./js/libs/trpLayout.js"></script>
parameter 설명
parameter 이름 parameter 설명
$tabBtn 텝버튼 선택자
$setctionSelector 섹션선택자
$sectionIDArr 섹션 아이디 배열
$activeCallbackFn($index, $state) $index : 인덱스
$state : 상태값 ("under", "underIn", "over", "overIN")
return method 설명
method 이름 method 설명
setScreenRatio( $ratio ) $ratio : (0~1) 메뉴 활성화 스크린위치 비율값
setScreenOffset( $offset ) 스크린 움직임 가감수치 (메뉴 높이 등....)
setGoScroll( $sectionID ) 섹션 아이디 값으로 이동
