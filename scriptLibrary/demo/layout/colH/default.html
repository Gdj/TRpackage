trpColH

col 아이템 높이 같게 하기

trpColH($col)

설명
trpUtility.js 불러오면 trpColH 사용할수 있습니다.
<script type="text/javascript" src="./js/libs/trpUtility.js"></script>
브라우져 리싸이즈를 체크하면 변화되는 컨텐츠 높이에따라 같은높이 값을 같게 한다.
class 설명
파라미터 이름 파마미너 설명
$col 한줄에 보여질 컨텐츠 겟수(필수)
class 설명
method 이름 method 설명
setCol 한줄에 보여질 컨텐츠 겟수 변경
getHeight 최대 높이값 가져오기
