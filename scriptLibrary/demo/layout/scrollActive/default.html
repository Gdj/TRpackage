trpScrollActive

타켓 위치에 도달하면 특정 엘리먼트에 클래스를 추가, 못미치면 삭제, 넘어가도 삭제

$("타겟선택자").trpScrollActive( $add_class, $show_per);

설명
trpLayout.js 불러오면 trpScrollPositionClass 사용할수 있습니다.
<script type="text/javascript" src="./js/libs/trpLayout.js"></script>
param 설명
파라미터 이름 파마미너 설명
$add_class 타겟 위치도달하면 추가될 클래스
$show_per 타겟이 상단 도착시`1.0` , 브라우저 절반 도착시 `0.5`
