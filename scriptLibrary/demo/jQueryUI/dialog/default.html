dialog

jQueryUI dialog로 대화창을 레어이로 띄워 줍니다.

.dialog( option )

설명
기본 위치값은 CSS 에서 설정해야합니다.
다이얼로그 옵션 정의
dialog 속성
매개변수 매개변수 설명
show: optiong 기본값"null"이며, 다이얼로그 보이게 할때 애니메이션 값을 타입에 따라 다르게 표현됩니다.
부울 값일때 : (false) 이면 애니메이션사용되지 않음, (true)이면 페이드 인아웃
숫자 값일때 : 페이드 되는 시간설정
문자 값일때 : "slideUp", "slideDown"등과 같은 내장 애니메이션
객체 값일대 : effect , delay , duration , easing 등과 같은 속성을 설정할수있음
hide: optiong 기본값"null"이며, 다이얼로그 숨기게 할때 애니메이션 값을 타입에 따라 다르게 표현됩니다.
부울 값일때 : (false) 이면 애니메이션사용되지 않음, (true)이면 페이드 인아웃
숫자 값일때 : 페이드 되는 시간설정
문자 값일때 : "slideUp", "slideDown"등과 같은 내장 애니메이션
객체 값일대 : effect , delay , duration , easing 등과 같은 속성을 설정할수있음
maxWidth 기본값"false"이며, 다이얼로그의 최대 픽셀 너비이값 지정
maxHeight 기본값"false"이며, 다이얼로그의 최대 픽셀 높이값 지정
minWidth 기본값(150)이며, 다이얼로그의 최소 픽셀 너비값 지정
minHeight 기본값(150)이며, 다이얼로그의 최소 픽셀 높이값 지정
autoOpen 기본값(true)이며, 대화상자가 자동으로 열림, (false)이면 열지지 않음.
closeText 기본값"close"이며, 닫기버튼에 title값을 변경합니다.
dialogClass 기본값""이며, 다이얼로그창에 추가적으로 클래스를 넣어 다른스타일을 적용할수있게 합니다.
draggable 기본값(true)이며, 제목 표시줄을 드래그할수있게 합니다. (false)이면 드래드되지 않습니다.
position 기본값:{ my: "center", at: "center", of: window }
my : 중심점을 이동시킬 수있습니다.
at : 중심점을 기준으로 정렬 합니다.
of : 대상을 기준으로 정렬 합니다.
resizable 기본값(treu)이며 다이얼로그 박스 크기조절 가능하도록 합니다.
close 기본값:(false) 이며, (ture)값이면 다이얼로그창 뒤에 오버레이어를 만들어 다이얼로그창왜에 접근이 되지 않도록 할수있습니다.
buttons 기본값:{} 이며, 다이얼로그창 밑부분에 버튼을 생성합니다.
dialog 메서드 설명
메서드 이름 설명 설명
close 다이얼로그 창을 닫습니다.
open 다이얼로그 창을 엽니다.
destroy 다이얼로그 창기능을 제거합니다.
dialog events
이벤트 이름 이벤트 설명
open 다이얼로그창이 열릴때 호출
open : function( event, ui ){}
close 다이얼로그창이 닫칠때 호출
close : function( event, ui ){}
focus 다이얼로그창에 포커스가 왔을때 호출
focus : function( event, ui ){}

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>jQuery UI Dialog - Default functionality</title>
	<link rel="stylesheet" href="../../../../asset/js/libs/jquery-ui-1.10.4.custom/themes/base/jquery.ui.all.css">
	<style type="text/css">
		/* 공통 스타일 */		
		body {	font-size: 62.5%; 	font-family: "Trebuchet MS", "Arial", "Helvetica", "Verdana", "sans-serif"; 	}
	</style>
	<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.11.1/jquery.min.js" ></script>	
	<script src="../../../../asset/js/libs/jquery-ui-1.10.4.custom.js"></script>
	<script>
	$(function() {
		$( "#dialog" ).dialog({
		  autoOpen: false,
		  show: {
			effect: "blind",
			duration: 1000
		  },
		  hide: {
			effect: "clip",
			duration: 500
		  },
		  closeText : "닫기",
		  dialogClass: "alert",
		  resizable : false,
		  modal: true,		  
		  buttons: {		
			yes: function() {
			  $( this ).dialog( "close" );
			},
			no: function() {
			  $( this ).dialog( "close" );
			}
		  }
		  		
		});
	 
		$( "#opener" ).click(function() {
		  $( "#dialog" ).dialog( "open" );
		});

		
  });
  </script>
</head>
<body>
 
<div id="dialog" title="Basic dialog">
  <p>This is an animated dialog which is useful for displaying information. The dialog window can be moved, resized and closed with the 'x' icon.</p>
</div>
 
<button id="opener">Open Dialog</button>
 
 
</body>
</html>
