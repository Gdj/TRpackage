Datepicker

jQueryUI Datepicker 를 간단한 옵션 값으로 제작할수 있습니다.

.datepicker( option )

설명
기본 위치값은 CSS 에서 설정해야합니다.
옵션 설정이 초기화때하는 방법과 초기화 이후 세팅방법이 틀리니 예제 소스를 잘보고 확인해주세요
Datepicker 설명
매개변수 매개변수 설명
option 하나 이상의 옵션을 설정 할때사용
dateFormat type:string
Default: "mm/dd/yy"
예제2) 참고
buttonImageOnly 버튼 요소로 랜더링 할지 아님 이미지로만 보일지 판단
showOn datepicker를 보이게 하는 옵션 (focus , button, both)
hide datepicker를 숨김
changeMonth "true" 달력에 "월"선택 박스생성
changeYear "true" 달력에 "년"선택 박스생성
defaultDate 기본적으로 선택되는 날짜 (년,월,일 혼용해서 사용가능)
Date표기방법 , Number(일) , String,등으로 표기할수 있습니다. defaultDate: "+ 7d" // 오늘부터 7일뒤
defaultDate: "- 1m" // 오늘부터 한달 전
defaultDate: "+ 1w" // 오늘부터 한주 뒤
defaultDate: "+ 1y" // 오늘부터 일련뒤
minDate 선택 제한 표현되는 날까지 선택되도록 함 (defaultDate)표현 방법이 같음
"+ 1w + 1d"
maxDate 선택 제한 표현되는 날까지 선택되도록 함 (defaultDate)표현 방법이 같음
"+ 2w + 1d"
numberOfMonths 표현되는 달력겟수 (숫자,배열로 선언)
firstDay 주가 시작되는 요일 (0:일요일, 1:월요일)
showWeek "true" 몇주 차인지를 보여줌
showButtonPanel "true" 버튼 패널 사용 여부
onClose DatePicker는 닫을 때 날짜가 선택되어 있는지의 여부에따라 호출된다.
onSelect DatePicker는이 선택 될 때 호출됩니다. 이 함수는 매개 변수로 텍스트와 DatePicker의 인스턴스로 선택한 날짜를받습니다.
