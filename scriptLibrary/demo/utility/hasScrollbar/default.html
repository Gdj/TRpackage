trpHasScrollBarX, trpHasScrollBar

스크롤바가 생성되었는지 boolean 값으로 리턴한다.

trpHasScrollBarX()

설명
trpUtility.js 불러오면 trpHasScrollBarX 사용할수 있습니다.
<script type="text/javascript" src="./js/libs/trpUtility.js"></script>
trpHasScrollBar 스크롤 Y 일때...
class 설명
속성 이름 속성 설명
this this 선택자에, "scrollbar_on"클래스 여부로 스크롤바가 생겼을때 스타일을 달리 줄수 있다.
return true(스크롤이 생성되었을때) | false(스크롤이 없을때)
