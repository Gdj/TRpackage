trpUtilityChangeOnceFnFn

값이 true, false일때 같은 값이 들어와도 1번만 실행해야 하는 경우 사용.

trpUtilityChangeOnceFnFn($functionTrue, $functionFalse)

설명
trpUtility.js 불러오면 trpUtilityChangeOnceFnFn 사용할수 있습니다.
<script type="text/javascript" src="./js/libs/trpUtility.js"></script>
class 설명
파라미터 이름 파라미터 설명
$functionTrue 값이 true 일때 실행될 함수
$functionFalse 값이 false 일때 실행될 함수
class 설명
메서드 이름 메서드 설명
setBoolean($boolean) true, false 값으로 각실행될 함수를 정한다.

<!DOCTYPE html>
<html>
<head>
	<meta charset="UTF-8">
	<style>

    .width_check { background-color: blueviolet; height: 50px; text-align: center;  font-size: 30px; font-weight: bold; color:bisque}
    .width_check.true {  background-color:cornflowerblue;}
        
	</style> 
	<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.11.1/jquery.min.js"></script>
	<script type="text/javascript" src="../../../../asset/js/libs/trpUtility.js"></script>
	<script>
	$(document).ready(function(){
        
        var onceFn = $(".width_check").trpUtilityChangeOnceFnFn(
            function(){ $(".width_check").addClass("true").text("width 768 over : true");     console.log(">> 768 over") }, 
            function(){ $(".width_check").removeClass("true").text("width 768 under : false"); console.log(">> 768 under") }
        );
		$(window).resize(function(){ 
            var win_width = $(window).width();
            if( win_width > 768){
                onceFn.setBoolean(true);
            }else{
                onceFn.setBoolean(false);
            }
        });
        $(window).trigger("resize");
        
	})
	</script>
	
</head>
<body>
	
  <div class="width_check"></div>
 
</body>
</html>
