trpBrowser, trpDevice

브라우저 이름과 버젼을 확인 할수 있습니다.

trpBrowser

설명
trpUtility.js 불러오면 trpBrowser를 사용할수 있습니다.
<script type="text/javascript" src="./js/libs/trpUtility.js"></script>
method 설명
메서드 이름 메서드 설명
name 현재 브라우저 이름 혹은 브라우저 벤터 이름을 불러 옵니다.
리턴되는 브라우저 이름 : [opr, chrome, safari, msie, mozilla]
version 현재 사용하고 있는 브라우저의 버젼을 불러 옵니다.

trpDevice

설명
trpUtility.js 불러오면 trpBrowser를 사용할수 있습니다.
<script type="text/javascript" src="./js/libs/trpUtility.js"></script>
method 설명
메서드 이름 메서드 설명
device 디바이스 출력 [ipad, tablet, android, iphone, pc ]
