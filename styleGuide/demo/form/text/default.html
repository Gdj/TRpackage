input [type="text"]

설명
input 요소의 text에 대한 재정의
attributes 옵션
속성 속성 설명
required 필수선택
pattern 정규식 패턴넣기
(예: pattern="^0\d{1,2}-?\d{3,4}-?\d{4}$") : 전화번호 정규식
(예: pattern="^[A-Za-z0-9._%+-]+@[A-Za-z0-9-]+(?:[.][A-Za-z0-9-]+)*[.][A-Za-z]{2,}$") : 이메일 정규식
css 옵션
css 속성 css 속성 설명
:valid 값이 유효할때
:invalid 값이 없거나 유효하지 않을때
::placeholder-shown:invalid placeholder 가있고 유효하지 않을때 : 첨에 값이없는데 유효하지 않는 표현을 피할때 사용
