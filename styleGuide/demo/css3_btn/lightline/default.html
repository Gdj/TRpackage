      - <number>	    실수(소수점 포함)입니다. 단위는 포함되지 않습니다.	1.5, 100, -0.5
      - <integer>	    정수 값입니다.	1, 10, -5
      - <angle>	      각도 값입니다. (이전 질문에서 사용됨)	45deg, 1.5rad, 0.5turn
      - <time>	      시간 단위 값입니다.	100ms, 1s
      - <percentage>	백분율 값입니다.	50%, 100%
      - <color>       색상 값입니다.	#ff0000, rgb(255, 0, 0), red

      initial-value: 0deg;
