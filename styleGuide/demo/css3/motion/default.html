CSS3 basic

transition 설명
transition: property name | duration | timing function | delay
transition: margin-left 4s ease-in-out 1s;
transition 속성 속성 설명
transition-property 트랜지션 이벤트를 적용할 CSS 속성
initial(defatul), none, all, inherit, css property명 (margin, width 등)
transition-duration 트랜지션 이벤트를 실행할 시간
0.3s(0.3초), 500ms(0.5초)
transition-timing-function 트랜지션 이벤트를 실행할 시간
linear : cubic-bezier(0, 0, 1, 1), ease-in-out : cubic-bezier(0.42, 0, 0.58, 1)
ease-in : cubic-bezier(0.42, 0, 1, 1), ease-out : cubic-bezier(0, 0, 0.58, 1)
transition-delay 트랜지션 지연 시간
0.3s(0.3초), 500ms(0.5초)
element.addEventListener('transitionend', EventHandlerTransitionEnd);

animation 설명
animation: name | duration | timing-function | delay | iteration-count | direction | fill-mode;
animation: animation_name .5s ease-in-out .2s infinite noraml forwards ;
animation 속성 속성 설명
animation-name 애니메이션 이름
@-webkit-keyframes 애니메이션이름 { 0% { top: 0 } 100% { top: 10px } 1}
animation-delay 애니메이션 실행 시간
animation-duration 애니메이션 지연 시간
animation-timing-function 이진 함수
animation-iteration-count 반복 횟수
1, 3, infinite
animation-direction 애니메이션 방향
noraml, revers(역재생), alternate(왔다갔다), alternate-reverse
animation-play-state 애니메이션의 동작 상태 여부
running = 재생중 | paused = 일시 정지중
animation-fill-mode 애니메이션의 실행 전/후에 대상에 스타일을 적용 방법
none (default) = 애니메이션 시작 전 요소에 할당된 스타일을 그대로 유지
forwards = 실행된 애니메이션의 마지막 keyframe 에 의해 설정된 스타일값을 유지
backwards = delay 가 일어나는 동안 keyframe 에 의해 설정된 시작 스타일값을 적용
both = forwards & backwards 가 동시에 적용
element.addEventListener("animationstart", EventHandler, false);
element.addEventListener("animationend", EventHandler, false);
element.addEventListener("animationiteration", EventHandler, false);

  }
