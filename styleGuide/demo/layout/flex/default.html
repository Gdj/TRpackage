display : flex

설명
css3 정열 방법 (wrap) (IE 11)
property 설명
컨테이너 property 이름 컨테이너 property 설명
flex-direction:[ row, column] 정열 방향 성 (중심선)
- row : 가로 방향 (↔)
- column : 세로 방향 (↕)
flex-wrap: [ nowrap, wrap, wrap-reverse] "flex-direction"속성이 "row" 일대 유연함 적용
- nowrap : 유연함 적용안함, 줄바굼 되지 않도록함.
- wrap : 유연함 적용, 유연하게 항목 줄바꿈
- wrap-reverse : 유연함 적용, "wrap" 반대로 줄바꿈
align-content: [ normal, center, flex-start, flex-end, space-around, space-between, stretch] "flex-direction" 중심 선 기준에따른 정열 방법, 기본(↕)
- center : 중앙 정열
- flex-start : 시작점에서 부터 정열
- flex-end : 끝나는점에서 부터 정열
- space-around : 중심으로부터 아이템에 같은 여백을 줘서 정열
- space-evenly : 아이템 간격이 동일하게 정열
- space-between : 중심 약끝으로 부터 정열
- stretch : 컨테이너 크기만큼 여백없이 정열
justify-content: [ normal, center, flex-start, flex-end, space-around, space-between, stretch] "flex-direction" 중심 선 반대 기준에따른 정열 방법, 기본(↔)
- center : 중앙 정열
- flex-start : 시작점에서 부터 정열
- flex-end : 끝나는점에서 부터 정열
- space-around : 아이템에 같은 여백을 줘서 정열
- space-evenly : 아이템 간격이 동일하게 정열
- space-between : 약끝으로 부터 정열
- stretch : 컨테이너 크기만큼 여백없이 정열
align-items: [ normal, center, flex-start, flex-end, stretch, baseline] 아이템 기준에서 정열 기준 (↕)
- center : 중앙 정열
- flex-start : 시작점에서 부터 정열
- flex-end : 끝나는점에서 부터 정열
- stretch : 컨테이너 크기만큼 여백없이 정열
- baseline : 텍스트기중
property 설명
컨테이너 아이템 property 이름 컨테이너 아이템 property 설명
flex :
[(px | %),
(flex-grow, flex-shrink, flex-basis)]		 단일 크기 지정
- (flex-grow, flex-shrink, flex-basis)
기본값 : 0 1 auto
flex-grow : number 아이템이 얼마나 커질지 비율을 정하는 속성
- 0 : 크기 증가 없음
- 1 : 크기 증가 함
- 2 : 크기 증가 함
flex-shrink : number 아이템이 얼마나 작아질지 비율을 정하는 속성
- 0 : 크기 증가 없음
- 1 : 크기 증가 함
- 2 : 크기 증가 함
flex-basis : (px | %) 초기 크기 지정
align-self : align-items : 설정 재정의
- auto : align-items 속성 값 사용
- stretch : 컨테이너 크기만큼 여백없이 정열
- center : 중앙 정열
- flex-start : 시작점에서 부터 정열
- flex-end : 끝나는점에서 부터 정열
- baseline : 텍스트기준 하단
order : number 플렉스 항목의 순서지정

   body, html { height: 100%; }
 
   .wrap { box-sizing: border-box; height: 50%; margin: 30px; 
     display: flex;
     /* 중심선 방향  (row : ↔) (column : ↕ ) */
     flex-direction: row;   
     flex-wrap: wrap;
     /*
       - center        : 중앙 정열
       - flex-start    : 시작점에서 부터 정열
       - flex-end      : 끝나는점에서 부터 정열
       - space-around  : 중심으로부터 아이템에 같은 여백을 줘서 정열
       - space-evenly  : 아이템 간격이 동일하게 정열 (justify-content)
       - space-between : 중심 약끝으로 부터 정열
       - stretch       : 컨테이너 크기만큼 여백없이 정열 (align-content) 
     */
     align-content: normal;     
     justify-content: flex-start;
     align-items: normal;
 
     padding: 10px;
     background-color: #f0f0f0;
   }
   /* 아이템 width는 최소값으로 반영된다고 보면 됩니다. */
   .wrap .item { padding: 20px; width: 30%; min-height: 30px; box-sizing: border-box;
     font-size: 20px; text-align: center;  border: 1px solid #333;
     background-color: #ddd; 
   }
 </style>
