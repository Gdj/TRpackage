display : flex

설명
css3 정열 방법 (wrap) (IE 11)
2차원적 구조를 가짐
property 설명
컨테이너 property 이름 컨테이너 property 설명
grid-auto-flow:: [row, column] 정열 방향
- row : 가로 방향 (↔)
- column : 세로 방향 (↕)
grid-template-columns: (px, %, fr, ...) 가로 크기 패턴 정의 (1fr 나머지크기, 모두 fr 이면 비율로 적용)
- grid-template-columns: 200px 200px 200px;
- grid-template-columns: repeat(3, 30%); - grid-template-columns: 200px 100px 1fr;
grid-template-row: (px, %, fr, ...) 가로 크기 패턴 정의 (1fr 나머지크기, 모두 fr 이면 비율로 적용)
- grid-template-row: 200px 200px 200px;
- grid-template-row: repeat(3, 30%); - grid-template-row: 200px 100px 1fr;
grid-auto-columns:: (px, %, auto, ...) 가로 크기지정 (auto 부족한 영역을 채워줌)
- grid-auto-columns: 200px;
- grid-auto-columns: minmax(100px, auto); (최소값, 최대값);
grid-auto-rows:: (px, %, auto, ...) 세로 크기지정 (auto 부족한 영역을 채워줌)
- grid-auto-rows: 200px;
- grid-auto-row: minmax(100px, auto); (최소값, 최대값)
grid-gap: (px, %, ...) 그리트 아이템 간격 (위아래, 좌우)
- grid-gap: 20px 10px;
align-content: [ normal, center, flex-start, flex-end, space-around, space-between, stretch] "flex-direction" 중심 선 기준에따른 정열 방법, 기본(↕)
- center : 중앙 정열
- flex-start : 시작점에서 부터 정열
- flex-end : 끝나는점에서 부터 정열
- space-around : 중심으로부터 아이템에 같은 여백을 줘서 정열
- space-evenly : 아이템 간격이 동일하게 정열
- space-between : 중심 약끝으로 부터 정열
- stretch : 컨테이너 크기만큼 여백없이 정열
justify-content: [ normal, center, flex-start, flex-end, space-around, space-between, stretch] "flex-direction" 중심 선 반대 기준에따른 정열 방법, 기본(↔)
- center : 중앙 정열
- flex-start : 시작점에서 부터 정열
- flex-end : 끝나는점에서 부터 정열
- space-around : 아이템에 같은 여백을 줘서 정열
- space-evenly : 아이템 간격이 동일하게 정열
- space-between : 약끝으로 부터 정열
- stretch : 컨테이너 크기만큼 여백없이 정열
align-items: [ normal, center, flex-start, flex-end, stretch, baseline] 아이템 기준에서 정열 기준 (↕)
- center : 중앙 정열
- flex-start : 시작점에서 부터 정열
- flex-end : 끝나는점에서 부터 정열
- stretch : 컨테이너 크기만큼 여백없이 정열
- baseline : 텍스트기중
property 설명
아이템 property 이름 아이템 property 설명
grid-area: name 아이템 이름 지어주기, 아래 내용에서 컬럼 번호 대신 이름으로 사용할 수 있다.
- grid-area: 이름
grid-area, grid-row-start, grid-column-start, grid-row-end, grid-column-end : (number, name) 번호 , 위치 - grid-area : grid-row-start grid-column-start grid-row-end grid-column-end
- grid-column : grid-column-start grid-column-end
- grid-row : grid-row-start grid-row-end

- grid-row-start : 2 | span 2; 시작위치 | 시작위치까지 채움
- grid-column-start : 2 | span 2; 시작위치 | 시작위치까지 채움
- grid-row-end : 3 | span 2; 끝위치 앞에까지 | 끝위치까지 채움
- grid-column-end : 3 | span 2; 끝위치 앞에까지 | 끝위치까지 채움
