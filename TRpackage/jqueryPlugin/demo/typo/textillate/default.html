textillate

텍스트 모션을 생성 textillate.js

.textillate({ option } )

설명
이미지로된 숫자를 호전하듯이 정열된는 효과.
관련 css가 선언되어야 합니다.
option
매개변수 매개변수 설명
selector 감지될 텍스트를 담아둘 선택자, 기본값은 "texts"
loop 반복실행 여부, 기본값은 false
minDisplayTime 각텍스트의 최소 모션 시간, 기본값은 2000
initialDelay 처음 실행될 지연 신간, 기본값은 0
autoStart 자동으로 실행될지에 대한 여부, 기본 값은 true
inEffects 보여지는 효과, 배열형태로 정의
outEffects 숨겨지는 효과, 배열형태로 정의 기본값은 ['hinge']
in:{} 보여지는 모션의 옵션 
    effect: 'fadeInLeftBig',    // 보여지는 효과      
    delayScale: 1.5,            // 각 문자에대한 지연 크기 
    delay: 50,                  // 각 문자에대한 지연 시간   
    sync: false,                // true 면 각문자가 동시에 모션됨 
    shuffle: false,             // 각문자가 렌덤으로 모션된지 여부 (sync:true 면 의미없음)
    reverse: false,             // 문자순서를 반대로 모션줄지 여부 (sync:true 면 의미없음) 
    callback: function () {}    // 보이는 모션이 끝나면 호출되는 함수
out:{} 숨겨지는 모션의 옵션 
    effect: 'hinge',            // 보여지는 효과     
    delayScale: 1.5,            // 각 문자에대한 지연 크기 
    delay: 50,                  // 각 문자에대한 지연 시간 
    sync: false,                // true 면 각문자가 동시에 모션됨 
    shuffle: false,             // 각문자가 렌덤으로 모션된지 여부 (sync:true 면 의미없음)
    reverse: false,             // 문자순서를 반대로 모션줄지 여부 (sync:true 면 의미없음)
    callback: function () {}    // 숨김 모션이 끝나면 호출되는 함수
callback 전체 모션이 끝나면 호출되는 함수
type 모션 줄단위, 기본값은 char('char' and 'word')
메서드
메서드 메서드 설명
$(element).textillate('start') 수동으로 모션 시작
$(element).textillate('stop') 수동으로 모션 멈춤
$(element).textillate('in') 수동으로 보여지는 모션 시작
$(element).textillate('out') 수동으로 숨겨지는 모션 시작
메서드
Events 이름 Events 설명
start.tlt textillate 모션 시작할때 호출
ex) $('start.tlt').on('inAnimationBegin.tlt', function () { //do something });
end.tlt textillate 모션 끝날때 호출
inAnimationBegin.tlt 보여지는 모션 시작할때 호출
inAnimationEnd.tlt 보여지는 모션 끝날때 호출
outAnimationBegin.tlt 숨겨지는 모션 시작할때 호출
outAnimationEnd.tlt 숨겨지는 모션 끝날때 호출
