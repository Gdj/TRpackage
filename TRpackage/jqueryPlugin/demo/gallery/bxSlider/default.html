bxSlider

플리킹 스라이딩 플러그인 bxSlider (모바일에서만 플리킹 됩니다.)

bxSlider(options)

설명
기본 위치값은 CSS 에서 설정해야합니다.
모바일에서만 터치가 가능합니다.
swiper 스크립트 : jquery.bxslider.min.js
bxSlider options 설명
options 속성 options 속성 설명
auto 기본 값은 "false" 이고 "true" 설정하면 자동재생 할 수 있습니다.
pause 기본 값은 4000 이고 자동재생 사이의 시간을 정의 할수 있습니다.
speed 기본 값은 500 모션 시간을 설정할수 있습니다.
autoDelay 기본 값은 0 자동플레이 지연 시간을 설정할수 있습니다.
startSlide 기본 값은 0 이고 처음 시작하는 인덱스 값을 설정합니다.
infiniteLoop 기본 값은 "true" 이고 "false" 설정하면 첨음과 마직막이 도달하면 넘어가지 않습니다.
pager 기본 값은 "true" 이고 "false" 설정하면 페이지를 숨길수 있습니다.
동적으로 태그및,클래스가 추가됩니다.
".bx-pager" 컨테이너로 ".bx-default-pager"하위 스타일 적용시 사용되며, 아이템(도트,페이지) 클래스는".bx-pager-item" 가생성됩니다.
pagerCustom 페이지를 커스텀으로 생성할수있게 합니다. 페이지를 생성할 wrap를 jquery 선택자형태로 선언합니다.
hideControlOnEnd 기본 값은 "false" 이고 "true" 설정하면 "다음"제어가 마지막 슬라이드와 그 반대에 숨겨집니다.
easing 기본 값은 "null" 이고 'linear', 'ease', 'ease-in', 'ease-out', 'ease-in-out', 'cubic-bezier(n,n,n,n)' 사용할수 있습니다.
adaptiveHeight 기본 값은 "false" 이고 "true"로 설정하면 동적으로 각 슬라이드의 높이에 따라 슬라이더 높이를 조정 합니다.
video 기본 값은 "false" 이고 "true"로 설정하면 슬라이드 비디오를 포함하는 경우로 설정, 또한, 포함 plugins/jquery.fitvids.js 참고
responsive 기본 값은 "true" 이고 "false"로 설정하면 외부 크기에 반응하지 않습니다.
preloadImages 기본 값은 "visible" 이고 "all"로 설정하면 이미지가 모드 로드되야 작동합니다.
preventDefaultSwipeX 기본 값은 "true" 이고 "false"로 설정하면 X축으로 이동하지 않습니다.
preventDefaultSwipeY 기본 값은 "false" 이고 "true"로 설정하면 Y축으로 이동합니다.
randomStart 기본 값은 "false" 이고 "true" 설정하면 랜텀으로 시작됩니다.
slideSelector 기본적으로 bxSlider 슬라이더 요소의 모든 직계 자식을 사용하며, 스라이드되는 요소를 jQuery 선택자처럼 선언합니다.
mode 기본 값은 "horizontal" 이고 "vertical", "fade"값을 사용할수 있습니다.
captions 기본 값은 "false" 이고 "true" 설정하면 이미지 "title"을 켑션으로 볼수있습니다.
slideWidth 슬라이더 요소의 넓이 "auto"슬아이더 크기
minSlides 보여지는 슬라이드 최소 갯수
slideMargin 슬라이드 되는 요소의 마진 수치
moveSlides 이동되는 슬라이드 요소 갯수
controls 기본 값은 "true" 이고 "false" 설정하면 좌우버튼을 생성하지 않습니다.
nextSelector 다음 버튼 지정 아이디값 or 클래스값
prevSelector 이전 버튼 지정 아이디값 or 클래스값
nextText 다음 버튼 요소에 ‘next’ 텍스트 삽입 , 텍스트가 필요 없다면 text-indent로 텍스트 숨김
prevText 이전 버튼 요소에 ‘prev’ 텍스트 삽입 , 텍스트가 필요 없다면 text-indent로 텍스트 숨김
bxSlider methods 설명
bxSlider 메서드 bxSlider 메서드 설명
.goToSlide(0) 인덱스 번호로 슬라이드로 실행 합니다.
.goToPrevSlide() 이전 슬라이드로 진행합니다.
.goToNextSlide() 다음 슬라이드로 진행합니다.
.startAuto() 자동슬라이드 시작 합니다.
.stopAuto() 자동슬라이드를 종료 합니다.
.getCurrentSlide() 현재 활성 슬라이드를 반환 합니다.
.getSlideCount() 슬라이드 총수를 반환 합니다.
.reloadSlider() 슬라이더를 다시로드합니다.
.destroySlider() (슬라이더를 호출하기 전에) 다시 원래 상태로 모든 슬라이더 요소를 되돌립니다.
Swiper event 설명
bxSlider 이벤트 bxSlider 이벤트 설명
onSliderLoad(currentIndex) 슬라이더가 완전히로드 된 직후 호출
onSlideBefore(($slideElement, oldIndex, newIndex)) 전환 직전에 호출
onSlideAfter($slideElement, oldIndex, newIndex) 각각의 슬라이드 전환 후 즉시 실행합니다. 함수 인수는 현재 슬라이드 요소 (전환이 완료 될 때)입니다.
onSlideNext($slideElement, oldIndex, newIndex) 각 "다음"슬라이드 전환 직전에 실행합니다. 함수 인수은 타겟 (다음) 슬라이드.
onSlidePrev($slideElement, oldIndex, newIndex) 각각의 "이전"슬라이드 전환 직전에 실행합니다. 함수 인수은 타겟 (이전) 슬라이드.

            slider.destroySlider();

                //controls : false,
                prevSelector : ".prev",
                nextSelector : ".next",
                auto: false,
                onSlideAfter: function($slideElement, oldIndex, newIndex){

                }
            })
        })
