.dotdotdot()

dotdotdot 이용하여 텍스트 넘침을 말줄임 한다.
link : dotdotdot(api)

.dotdotdot( { options } )

설명
기본 위치값은 CSS 에서 설정해야합니다.
options 설명
options 이름 options 설명
ellipsis 줄임표로 추가할 텍스트 기본값은 "..."
wrap 텍스트를 자르는 방법, 기본값은 "word" ['word','letter','children']
fallbackToLetter 긴 단어의 경우 'letter'로 대체 옵션 전환, 기본값 true
after 줄임표 다음에 유지하고 넣을 요소의 선택 자, 기본값 null
watch 줄임표 업데이트 여부, 기본값 false, [false, true.,'window']
height 최대 높이를 설정할수있고, 기본값은 null
tolerance 높이 옵션의 편차, 기본값 0
callback function (isTruncated, orgContent) {}
lastCharacter 잘린 텍스트의 문자를 제거,
lastCharacter { remove [ ' ', ',', ';', '.', '!', '?' ] }
ellipsis 배열문자 포함시 줄임표시 않함, []
event 설명
event 이름 event 설명
update $("selectName").trigger("update",function(){});
isTruncated var isTruncated = $("selectName").triggerHandler("isTruncated");
originalContent var content = $("selectName").triggerHandler("originalContent");
destroy $("#wrapper").trigger("destroy",function(){});

            var _dots = $('.box.after');
            
            _dots.append( ' <a class="toggle" href="#"><span class="open">[ + ]</span><span class="close">[ - ]</span></a>' );
            
            function createDots(){
                _dots.dotdotdot({
                    after: 'a.toggle',
                    watch: 'window'
                });
            }
            function destroyDots() {
                _dots.trigger( 'destroy' );
            }
            createDots();

            _dots.on(
                'click',
                'a.toggle',
                function() {
                    _dots.toggleClass( 'opened' );

                    if ( _dots.hasClass( 'opened' ) ) {
                        destroyDots();
                    } else {
                        createDots();
                    }
                    return false;
                }
            );
            
            
        });
	</script>
