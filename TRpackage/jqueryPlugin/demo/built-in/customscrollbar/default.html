mCustomScrollbar

mCustomScrollbar를 이용하여 다양한 스크롤바를 만들수있다.
link : mCustomScrollbar

.mCustomScrollbar( { option } )

설명
기본 위치값은 CSS 에서 설정해야합니다.
버튼 옵션 정의
autoHideScrollbar : true
mCustomScrollbar 옵션 설명
매개변수 매개변수 설명
setWidth 컨텐츠 폭설정 정수(픽셀), 문자열(비율)
setHeight 컨텐츠 높이설정 정수(픽셀), 문자열(비율)
setTop 컨텐츠 위치 top설정 정수(픽셀) 만큼 스크롤
setLeft 컨텐츠 위치 left설정 정수(픽셀) 만큼 스크롤
axis 스크롤 방향 "y"가 기본값 (y, x, yx)
scrollbarPosition 스크롤 생성위치 "inside"기본 값 (inside, outside )
scrollInertia 애니메이션 지속 시간 (초)
autoDraggerLength 스크롤 바를 자동 조절할때 "true",
스크롤 바를 고정크기로 만들때 "false"
autoHideScrollbar 기본값 : false
"true" 값이면 컨텐츠에 오버되었을때 스크롤생김
autoExpandScrollbar 기본값 : false
"true" 값이면 스크롤에 오버되었을때 확장됨.
theme 테마적용 : 기본값 "light"
"light" // default, "dark", "minimal", "minimal-dark", "light-2", "dark-2", "light-3", "dark-3", "light-thick", "dark-thick", "light-thin", "dark-thin", "inset", "inset-dark", "inset-2", "inset-2-dark", "inset-3", "inset-3-dark", "rounded", "rounded-dark", "rounded-dots", "rounded-dots-dark", "3d", "3d-dark", "3d-thick", "3d-thick-dark"
mCustomScrollbar 메서드 설명
메서드 메서드 설명
scrollInertia 스크롤 시간 정수로 변경
scrollEasing 스크롤 인진값
"linear", "easeOut", "easeInOut"
update 수동으로 스크롤바업데이트
scrollTo 특정 위치로이동
"string"
e.g. element selector: "#element-id"
e.g. special pre-defined position: "bottom"
e.g. number of pixels less/more: "-=100"/"+=100"
integer
e.g. number of pixels: 100
[array]
e.g. different y/x position: [100,50]
object/function
e.g. jQuery object: $("#element-id")
e.g. js object: document.getelementbyid("element-id")
e.g. function: function(){ return 100; }
stop 진행중인 애니메이션을 중지
disable 스크롤을 일지적으로 비활성화 (false,true)
destroy 사용자 스크롤을 제거
mCustomScrollbar callbacks 이벤트 설명
callbacks 이벤트 callbacks 이벤트 설명
onCreate 마크업이 생성될때 호출
onInit 스크롤바가 초기화될때 호출
onScrollStart 스크롤이 시작될때 호출
onScroll 스크롤이 멈추면 호출
onUpdate 스크롤이 갱싱되면 호출
onImageLoad 이미지가 로들될때마다 onUpdate호출
