tableHeadFixer

link : tableHeadFixer Examples

.tableHeadFixer( { options } )

설명
테이블 부모의속성에 스크롤 속성이 있어야 합니다.
options 설명
options 이름 options 설명
head [true] thead 고정이 기본 값.
foot [false] tboot 고정 시킬지 여부.
left 왼쪽에 고정시킬 컬럼수
right 오른쪽에 고정시킬 컬럼수
