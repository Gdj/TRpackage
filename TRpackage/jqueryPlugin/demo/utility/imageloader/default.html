imageloader

beatak jquery-imageloader을 이용하여 이미지로드 합니다.
GitHub API : beatak imageloader jquery Plugin(GitHub API) GitHub Download : beatak imageloader jquery Plugin(GitHub)

.imageloader( { option } )

설명
기본 위치값은 CSS 에서 설정해야합니다.
아이템의 속성값으로 "data-src"만들어 실제 이미지 경로를 넣어줍니다.
imageloader
옵션 옵션 설명
selector 컨테이너 안에 선택할때 사용
dataattr 기본값"src" = "data-src" 이미지경로
background 기본값은 false 값이고, true로 설정하면 배경으로 이미지를 로드할수 있습니다.
imageloader이벤트
옵션 함수 옵션 함수 설명
each 여러개의 이미지를 불러올경우 각각의 아이템이 첫번째 인수로 넘어옵니다.
callback 모두 로드되면 컨테이너를 콜백함수로 첫번째인수가 this로 전달됩니다.

      body {  font-size: 62.5%;   font-family: "Trebuchet MS", "Arial", "Helvetica", "Verdana", "sans-serif";     }

      .lazy-img {  width: 150px; height: 150px; float: left; }
    </style>

    <script type="text/javascript">
      $(function() {

        $("#load_btn").on("click", function() {
          console.log("start : " + $(".lazy-img").height());
          $('.example').imageloader({
            selector: '.lazy-img',
            each: function(elm) {
              console.log(elm + "each : " + $(elm).height());
            },
            callback: function(elm) {
              $(elm).slideDown();
              console.log("end : " + elm + $(elm).height())
            }
          }); // imageloader
          console.log("clickend : " + $(".lazy-img").height());
        }); // click	

      });
  </script>
