meta 자동링크 막기

HTML 모바일 전화번호 자동링크 막기

<meta name=”format-detection” content=”telephone=no” />

HTML 모바일 전화, 주소, 이메일 자동링크 막기

<meta name=”format-detection” content=”telephone=no, address=no, email=no” />
