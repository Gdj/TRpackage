CSS 가이드

기본 규칙

css 데이터를 로딩할 경우 @import 방식은 사용하지 않는다. (일부 브라우저에서 이미지 로딩 후 적용되기 때문에)

external 방식에서는 charset을 표기한다.

@charset "utf-8"

속성 선언 순서

css 속성 선언 순서
순서 의미 대표되는 속성 관련 속성
1 흐름 float float, clear
2 표시 display visibility, (width, height, margin, padding)
3 위치 position position, (top & bottom), (left & right)
4 내용 font line-height, vertical-align, text-align, text-indent, font-weight, font-size, color, letter-spacing, text-decoration, white-space 등
5 넘침 overflow -
6 배경 background -
7 테두리 border -
8 기타 - z-index, *zoom

순서 { 흐름; 표시; 크기; 간격; 위치; 내용; 넘침; 배경; 테두리; 기타; }

예제

{ float:left; display:block; width:500px; height:50px; margin:0; padding:0; line-height:0px;
text-align:center; text-indent:0px; font-weight:bold; font-size:12px; color:#ddd; letter-spacing:-1px; overflow:hidden;
white-space:nowrap; background:url('../images/icon.gif') no-repeat center top; border:1px solid #eee;
position:absolute; top:0; left:0; border-top:1px solid #ccc; z-index:10; *zoom:1; }
