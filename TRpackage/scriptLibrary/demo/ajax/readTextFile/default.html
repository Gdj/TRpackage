trpReadTextFile

ajax XMLHttpRequest 통신으로 파일을 불러옵니다.

trpReadTextFile($file, $callback)

설명
trpAjax.js 불러오면 trpAjax, 를 사용할수 있습니다.
<script type="text/javascript" src="./js/libs/trpAjax.js"></script>
파일을 읽어와 json 형태로 파싱(Parsing)
class 설명
parameter 이름 parameter 설명
$file 읽어올 파일 경로이름
$callback 파일읽으면 실행될 함수

<body>
  
  <div id="json_data"></div>
  
  <script>
    
    trpReadTextFile("./json_data.json", function(text){
      var mydata = JSON.parse(text);
      console.log(mydata.books);

      mydata.books.forEach(function($item, $index){
        document.getElementById("json_data").innerHTML += $index + ": id    : " + $item.id + "<br>";
        document.getElementById("json_data").innerHTML += $index + ": name  : " + $item.name + "<br>";
      });
    });
  </script>
