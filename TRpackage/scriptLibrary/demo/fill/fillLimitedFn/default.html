trpFillLimitedFn

브라우져, 트리거 타겟 크기로 화면을 채움.

trpFillLimitedFn(parameter)

설명
trpFill.js 불러오면 trpFillLimitedFn, 를 사용할수 있습니다.
<script type="text/javascript" src="./js/libs/trpFill.js"></script>
가로세로중 작은 크기에 비례하여 채워집니다.
windows resize 이벤트를 생성해서 사용합니다.
최소 값을 설정할수 있습니다.
class 설명
parameter 구분 parameter 설명
$container 타겟 컨테이너 (필수)
$image 타겟 컨텐츠 (필수)
$targetWidth 타겟 가로 크기 (필수)
$targetHeight 타겟 세로 크기 (필수)
$triggerWidth width가 참조될 (필수)
$triggerHeight height가 참조될 (필수)
$hearderHeight 상단 마진값 (필수)
$footerHeight 하단 마진값 (필수)
