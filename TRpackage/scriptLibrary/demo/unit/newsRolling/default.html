trpNewsRolling

아래에서 위로 올라가는 news 롤링 (정지, 시작 기능)

trpNewsRolling($time)

설명
참고값에 변경하고자 하는 element를 선택해 사용.
<script type="text/javascript" src="./js/libs/trpUnit.js"></script>
parameter 설명
parameter 이름 parameter 설명
$time 슬라이드 지연 시간 (default:3000)
method 설명
method 이름 method 설명
next() 다음 슬라이드
prev() 이전 슬라이드
play() 플레이 시작 (timer start)
stop() 플레이 멈춤 (timer stop)
