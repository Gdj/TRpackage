trpRatioSize

trpRatioSize 메소드를 이용해 가로크기 비율에따른 타겟값의 크기(px)를 변경합니다.

trpRatioSize( {options} )

설명
참고값에 변경하고자 하는 element를 선택해 사용.
<script type="text/javascript" src="./js/libs/trpUnit.js"></script>
속성 설명
options 이름 options 설명
maxTri : 기본값은 9999 이며,
참조가 되는 width값의 최대 값으로 작아지는 경우에만 사용할경우 참고대상의 width값을 넣는다.
minTri : 기본값은 1 이며,
참조가 되는 width값의 최소 값으로 작아지는 경우에 사용할경우 생략 가능하다.
maxTar : 기본값은 9999이며,
변화될 타겟 값의 최대 값으로 작아지는 경우에만 사용할경우 타겟값의 최대크기를 넣는다.
minTar : 기본값으 1이며,
변화될 타겟 값의 초소값으로 더이상 줄어들지 않을값을 넣으며 비율대로 줄어들경우 생략 가능하다.
property : 기본값은 "height"이며,
참조되는 값의 width값에 비례하여 타겟이 되는 css속성값을 변경한다.
"font-size"를 넣으면 참고되는 값에 비례하여 폰트 크기를 변경할수있다.
