trpScrolltop

trpScrolltop($tar, $add)

설명
trpUtility.js 불러오면 trpRemoveClassPrefix 사용할수 있습니다.
<script type="text/javascript" src="./js/libs/trpUtility.js"></script>
class 설명
파라미터 이름 파마미너 설명
$tar 타켓위치로 이동 (필수)
$add 가감 수치 (선택)
