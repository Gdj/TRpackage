trpClassBubbleCheck

클릭 이벤트로 타겟 대상의 부모클래스가 있는지를 체크

trpClassBubbleCheck($className, $activeFn)

설명
trpUtility.js 불러오면 trpClassBubbleCheck를 사용할수 있습니다.
<script type="text/javascript" src="./js/libs/trpUtility.js"></script>
class 설명
파라미터 이름 파마미너 설명
$className 검색할 부모 클래스 이름
activeFn(boolean) 클래스 이름이 있으면 true, 없으면 false 를 넘김
