trpShuffle($arr)

설명
배열을 넘기면 랜덤하게 섞어서 리턴
class 설명
파라미터 이름 파마미너 설명
$arr 섞일 배열

trpArrTper($arr)

설명
배열의 최대값으로 각각의 백분율 퍼센트배열 리턴 (소숫접 2재짜리까지)
class 설명
파라미터 이름 파마미너 설명
$arr number 배열값

trpMaxTper($arr, $max)

설명
최대값으로 각각의 백분율 퍼센트배열 리턴 (소숫접 2재짜리까지)
class 설명
파라미터 이름 파마미너 설명
$arr number 배열값
$max number 최대값 : 100%

trpArrMax($arr)

설명
배열중 최대값 리턴
class 설명
파라미터 이름 파마미너 설명
$arr number 배열값

trpArrMaxIndex($arr)

설명
배열중 최대값의 Index를 리털
class 설명
파라미터 이름 파마미너 설명
$arr number 배열값

trpArrAve($arr)

설명
배열 평균값을 리턴
class 설명
파라미터 이름 파마미너 설명
$arr number 배열값

trpArrSum($arr)

설명
배열 총합을 리턴
class 설명
파라미터 이름 파마미너 설명
$arr number 배열값
