trpTip

툴팁 생성 :
기본적으로 target의 중앙 하단에 생성되며, (좌,우) 가 넘어가면 각각 (좌,우)정렬 됨.
하단영역이 없으면 target 상단에 생성.

trpTip([options])

설명
trpFrom.js 불러오면 trpTip, 를 사용할수 있습니다.
<script type="text/javascript" src="./js/libs/trpFrom.js"></script>
options 설명
options 이름 options 설명
tipArea 툴팁 최상단 컨테이너
tipWrap 툴팁 pop 선택자
method 설명
method 이름 method 설명
open(this) 툴팁이 열립니다.
this : 툴팁 대상
close() 툴팁이 닫침니다.
position(this) tooltip의 위치를 갱신합니다.
this : 툴팁 대상
resetEvent(this) mouseover, mouseout 이벤트를 삭제 합니다. br
this : 툴팁 대상
