trpFilestyle

trpFilestyle 메소드를 이용해 파일 선택버튼을 이미지로 변경 할수있는 jquery 플러그인 입니다.

trpFilestyle( {options} )

설명
trpFilestyle 메서드를 사용하면 필요한 태그가 생성됩니다.
<script type="text/javascript" src="./js/libs/trpForm.js"></script>
class 설명
클래스 이름 클래스 설명
trp_fileName <input type="file" /> 태그를 감싸는 div태그의 class 이름입니다.
스타일을 제정의 하려면 "!important" 사용합니다.
trp_fileButton input type="file"을 감싸고있는 div의 클래스 이름입니다.
속성 설명
options 이름 options 설명
textClass: 파일을 선택하면 파일경로가 표현되는 "input"태그의 클래스이름을 설정합니다.
기본값으로 "trp_fileName"설정되어 있으며 필요에따라 변경할수 있으며 상세한 스타일은 클래스이름으로 재정의 합니다.
textWidth: 파일을 선택하면 파일경로가 표현되는 "input"태그의 width값을 설정합니다.
textHeight: 파일을 선택하면 파일경로가 표현되는 "input"태그의 height값을 설정합니다.
buttonClass: input type="file"을 감싸고있는 "div"태그의 클래스이름을 설정합니다.
기본값으로 "trp_fileButton"설정되어 있으며 필요에따라 변경할수 있으며 상세한 스타일은 클래스이름으로 재정의 합니다.
buttonText: input type="file"을 감싸고있는 div의 버튼이름을 정의할 수 있습니다.
기본값은"파일선택"으로 되어있고 변경할수 있습니다. 단,"buttonImage" 속성값이 없을때만 적용됩니다.
buttonImage: input type="file"을 감싸고있는 div의 버튼을 이미지로 변경할때의 이미지 경로를 넣어줍니다.
기본값은 ""이며 값을정의하지 않으면 "buttonText"속성 값이 적용됩니다.
buttonWidth: input type="file"을 감싸고있는 div의 버튼의 width값을 정의합니다.
기본값은 "auto"이며 버튼 이미지 width값으로 설정합니다.
buttonHeight: input type="file"을 감싸고있는 div의 버튼의 height값을 정의합니다.
기본값은 "auto"이며 버튼 이미지 height값으로 설정합니다.
