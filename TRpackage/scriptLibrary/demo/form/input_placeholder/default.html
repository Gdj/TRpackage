placeholder 속성 적용

placeholder 속성이 없는 ie7,8,9 버젼에서도 효과를 줄수 있도록 한 스크립트 입니다.

placeholder

설명
trpForm.js 불러오면 placeholder 효과가 적용 됩니다.
<script type="text/javascript" src="./js/libs/trpForm.js"></script>
class 설명
클래스 이름 클래스 설명
trp_inputWarp input 태그를 감싸는 div태그의 class 이름입니다.
스타일을 제정의 하려면 "!important" 사용합니다.
trp_placeholder input 태그 앞에 span태그의 class 이름입니다.
placeholder속성이 지원되지 않은 브라우저에 가상으로 태그를 넣어줍니다.
스타일을 제정의 하려면 "!important" 사용합니다.
