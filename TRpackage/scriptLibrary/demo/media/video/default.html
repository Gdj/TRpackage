video script 컨트롤 방법

video

설명
비디오 태그에 외부 컨트롤러 적용하기
Media properties AIP
Media events AIP

태그 속성

video option
video option 이름 video option 설명
autoplay 동영상이 준비되는 즉시 재생을 시작하도록 지정합니다. 적용값(autoplay)
controls 비디오 컨트롤이 표시되도록 지정합니다 (예 : 재생 / 일시 중지 단추 등)
controlsList="nodownload" `controls` 설정시 `controlsList="nodownload"`옵션을 주면 다운로드를 막는다.
width 비디오 플레이어의 넓이(px)를 설정합니다
height 비디오 플레이어의 높이(px)를 설정합니다
loop 비디오가 끝날 때마다 다시 처음부터 다시 시작하도록 지정합니다.
muted 비디오의 오디오 출력을 음소거를 지정합니다.
poster 비디오가 다운로드되는 동안 또는 사용자가 재생 버튼을 누를 때까지 표시 할 이미지경로를 지정합니다.
preload 제작자가 페이지가로드 될 때 비디오를로드해야한다고 생각하는지 여부 및 그 방법을 지정합니다.
auto(자동으로 동영상을 미리 준비),
metadata(크기, 첫 프레임, 트랙 리스트, 길이 등을 미리 로드),
none(아무것도 하지않음) 중에 선택하면 됩니다.
source option
source option 이름 source option 설명
src 비디오 파일의 URL을 지정합니다.
type 미디어 자원의 유형을 지정.
"video/mp4", "video/ogg", video/webm,
"audio/mpeg", audio/mp4, "audio/ogg", "audio/webm", "audio/wav"
media 미디어 자원의 유형을 지정
"media_query"

스크립트 속성

properties
properties 이름 properties 설명
.autoplay 자동 재생여부 boolean값.
.currentSrc 선택된 미디어 자원의 절대 URL을 포함한 DOMString를 돌려줍니다.
.initialTime 초의 초기 재생 위치를 나타내는 double 값을 반환합니다.
.currentTime 현재 재생 시간을 초 단위로 나타내는 double 값입니다. 이 값을 설정하면 미디어가 새 시간으로 검색됩니다.
.duration 매체의 길이를 초 단위로 나타내는 double 값을 리턴합니다. 매체 데이터가 없으면 0을 리턴합니다.
.ended 미디어 요소의 재생이 완료되었는지 여부를 나타내는 부울을 반환합니다.
.loop 미디어 요소가 끝에 도달 할 때 미디어 요소를 다시 시작해야하는지 여부를 나타내는 루프 HTML 특성을 나타내는 부울입니다.
.muted 오디오가 음소거되는지 여부를 결정하는 부울입니다.
오디오가 음소거인 경우는 true, 그렇지 않은 경우는 false
.paused 미디어 요소가 일시 중지되었는지 여부를 나타내는 부울을 반환합니다.
.playbackRate 미디어가 재생되는 속도를 나타내는 double 값입니다.
methods
methods 이름 methods 설명
.canPlayType() 지정된 미디어 유형을 재생할 수 있는지 여부를 결정합니다.
.fastSeek() 주어진 시간을 직접 찾습니다.
.load() 미디어 요소를 재설정하고 미디어 리소스를 다시 시작합니다.
.pause() 미디어 재생을 일시 중지합니다.
.play() 미디어 재생을 시작합니다.
Media events
Media events 이름 Media events 설명
loadstart 미디어가 로드 시작할대.
play 일시 정지 된 후 미디어 재생이 시작될 때 전송됩니다
pause 재생이 일시 정지되면 보냅니다.
progress 미디어를 다운로드하는 진행 상황을 이해 관계자에게 알리기 위해 주기적으로 전송됩니다
ended 재생이 완료되면 전송됩니다.
error 오류가 발생하면 전송됩니다. 요소의 error속성에는 자세한 정보가 들어 있습니다. 자세한 내용은 HTMLMediaElement.error 를 참조하십시오.
