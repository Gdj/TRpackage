trpToggleWrapBtn

클릭할때마다 반대 형태로 변하는 버튼

.trpToggleWrapBtn($trueFn, $falseFn)

설명
기본 위치값은 CSS 에서 설정해야합니다.
"data-active" 속성값을 생성하여 true, false 값을 변경하는 원리 입니다.
toggle btn 설명
매개변수 매개변수 설명
$trueFn true 상태일때 실행되는 함수
$falseFn false 상태일때 실행되는 함수
tab menu 설명
메서드 메서드 설명
toggleAllSet( ture | false ) 전체 메뉴를 활성화, 비활성화 합니다.
setToggle( index ) 활성화 인덱스를 넣으면 해당 버튼만 토글
setBtn( index ) 활성화 인덱스를 넣으면 해당 버튼만 "true" 상태로 바꿈.
reSetBtn( index ) 활성화 인덱스를 넣으면 해당 버튼만 "false" 상태로 바꿈.
setOneBtn( index ) 활성화 인덱스를 넣으면 해당 버튼만 "true" 나머지 버튼은 "false" 상태로 바꿈.

    $(document).ready(function(){
      var btnControl;
      btnControl =  $(".toggle-wrap").trpToggleWrapBtn( ".toggle0 > a",
        function( $btnAll ){				
          $($btnAll, ".toggle-wrap").addClass("on");
          $($btnAll, ".toggle-wrap").parent().find(".con").stop().slideDown("fast");		
        },function( $btnAll ){		
          $($btnAll, ".toggle-wrap").removeClass("on");
          $($btnAll, ".toggle-wrap").parent().find(".con").stop().slideUp("fast");
        }
      )

      $(".btn_off").on("click"	 , function(){ btnControl.toggleAllSet(false); });
      $(".btn_on").on("click"	 , function(){ btnControl.toggleAllSet(true);  });

      $(".toggle").on("click"	 , function(){ btnControl.setToggle(3);  });
      $(".btn_set").on("click"	 , function(){ btnControl.setBtn(2);     });
      $(".btn_reset").on("click" , function(){ btnControl.reSetBtn(2);   });
      $(".btn_oneset").on("click", function(){ btnControl.setOneBtn(1);  });

    })

  </script>
