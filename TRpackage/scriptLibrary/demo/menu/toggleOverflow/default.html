trpToggleOverflow

토글버튼과 컨네이너 영역을 벗어나면 해제 (select box)재현

.trpToggleOverflow($containerSelerter, $delay)

설명
기본 위치값은 CSS 에서 설정해야합니다.
body 크기안에 서만 이벤트가 발생 함으로 최소 크기가 window.height 여야 함.
trpToggleOverflow 설명
매개변수 매개변수 설명
$containerSelerter 컨테이너 영역 밖을 클릭시 해제
$delay 지연시간
tab menu 설명
메서드 메서드 설명
$firstFn() 첫번째 실행 함수 (실행)
$secondFn() 두번째 실행 함수 (취소)
