draggable

jQuery draggable로 드래그 가능한 요소로 만듬니다.

.draggable( option )

설명
기본 위치값은 CSS 에서 설정해야합니다.
옵션 설명
매개변수 매개변수 설명
scroll 기본값 : true 이며, "false"로 설정하면 화면밖으로 드래그하면 자동으로 스크롤 되지 않는다
scrollSensitivity 기본값 : 20 , 자동스크롤이 시작되는 거리를 화면안쪽 지정된 크기에서 부터 시작됩니다.
("scroll:false" 설정이면 실행되지 않습니다.)
scrollSpeed 기본값 : 20 , 스크롤 되는 속도를 조절합니다.
("scroll:false" 설정이면 실행되지 않습니다.)
axis 기본값 : false , 드래그 방향을 제한 합니다.
  • "x" 값이면 x축으로만 드래드 가능합니다.
  • "y" 값이면 y축으로만 드래가 가능합니다.
containment 기본값 : false , 드래그 영역을 제안 합니다.
  • 선택자 : 선택자 크기 안에서 드래그 되도록 합니다.
  • 문자열 : "parent", "document", "window" 값을 사용할수있습니다.
  • 배열 :[ x1, y1, x2, y2 ] 경계면을 배열형태로 정의 할수있습니다.
cursor 기본값 : auto , 커서 모양을 변경합니다.
  • 문자열 : "hand", "pointer", "move" 등의 값을 사용할수있습니다.
cursorAt 기본값 : false , 드레그 타겟의 오프셋을 설정합니다.
{ top: 56, left: 56 } 드레그 타겟기준으로 오프셋을 설정
handle 기본값 : false , 드래그 타겟을 선택자만 드레그되도록 변경
cancel 기본값 : false , 드레그 타겟의 오프셋을 설정합니다.
{ top: 56, left: 56 } 드레그 타겟기준으로 오프셋을 설정
Methods 설명
메서드 이름 메서드 설명
disable 드래그가 되지않게 설정 합니다.
enable 드래그가 가능하도록 설정 합니다.
events 설명
이벤트 이름 이벤트 설명
drag 드래드 되는 동안 호출
function( event, ui )
start 드래스 시작할때 호출
function( event, ui )
stop 드래그 멈추었을때 호출
function( event, ui )
