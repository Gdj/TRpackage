button

jQuery button style 을 태그와 관계없이 동일한 스타일을 적용시킬수 있습니다.

.button( option ) , .buttonset( option )

설명
기본 위치값은 CSS 에서 설정해야합니다.
버튼 옵션 정의
button 설명
매개변수 매개변수 설명
disabled 기본값"false" 이면 ,"true"이면 버튼을 비활성화시킴
icons : { primary:UI클래스 , secondary:UI클래스 } "primary" 버튼 앞쪽에 넣을 아이콘, secondary 버튼 뒤쪽에 넣을 아이콘
