window Popup

window Popup 버튼을 클릭하면 window Popup 열리고 닫기 버튼으로 팝업을 닫고 링크를 관리할 수 있다.
<a href="이미지경로" onclick="window.open(this.href, '_blank','width=1280px, height=720px, toolbars=no, scrollbars=yes'); return false;"> </a>

.trpWindowPopup( url , width, height, top, left )

설명
팝업창 여는 부모
기본 위치값은 CSS 에서 설정해야합니다.
window Popup 설명
매개변수 매개변수 설명
url 팝업 창열릴 주소 (필수)
width 팝업 창 가로 크기 (필수)
height 팝업 창 세로 크기 (필수)
top 팝업 창열릴 세로위치
left 팝업 창열릴 가로위치
window Popup 설명
메서드 메서드 설명
oneDay(cookieName) 오늘하루 팝업열지않기에 따라 팝업 제어,
cookieName(설정할 쿠키이름),
팝업창에서 설정한 쿠키이름과 같아야합니다. (필수)
윈도우팝업
설명 직접 사용 코드
열기 onClick="popWin = window.open('http://www.naver.com','windows','width=800,height=900,scrollbars=no'); popWin.focus(); return false;"
닫기 self.close();
window.close();

.trpWindowClose()

설명
팝업창 닫을때 사용하며 링크에 대한 옵션을 선택할수 있습니다.
기본 위치값은 CSS 에서 설정해야 합니다.
window Close 설명
메서드 메서드 설명
parentLink(url) 부모창에서 'url'링크주소로 열기고 현제창을 닫침니다.
selfLink(url) 현제창에서 'url'링크가 열리기 때문에 창은 닫치지 않습니다.
blankLink(url) 새창으로 'url'링크주소로 열기고 현제창을 닫침니다.
oneDay(cookieName) 오늘하루 팝업열지않기에 따라 팝업 제어,
cookieName(설정할 쿠키이름),
부모창에서의 쿠키이름과 같아야합니다. (필수)

.trpWindowPopupResize()

설명
팝업창을 특정 선택자 크기에 맞게 열리도록 합니다.
기본 위치값은 CSS 에서 설정해야 합니다.
