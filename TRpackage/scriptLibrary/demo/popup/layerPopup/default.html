layer Popup

layer Popup 버튼을 클릭하면 layer Popup 열림

.trpLayerPopup( options )

설명
기본 위치값은 CSS 에서 설정해야합니다.
options 설명
options 이름 options 설명
popupTarget 팝업 위치를 위한 선택자. 기본값: ".trp_popup"
기본값으로 popupTarget이 중앙정열 됩니다.
popupWarp 팝업 레이어를 감싸고있는 선택자.
팝업을 숨기거나 보이게할 선택자입니다. 이 안에는 bgDim요소와 popupTarget이 들어 올 수 있습니다.
값을 넣지 않으면 a테그 "herf" 타겟값으로 설정됩니다.
layer Popup 설명
메서드 메서드 설명
align($top, $left) popupTarget 위치변경
$top : 정열 상단 위치값, ("center")로입력하변 중앙 정열됩니다.
$left : 정열 왼쪽 위치값, ("center")로입력하변 중앙 정열됩니다.
기본 값은 중앙 정열 이며, "center"라는 값이 있어도 resize 이벤트가 실행됩니다.
bgDim($target, $opacity, $bgColor ) 팝업 전체 영역에 배경을 넣어 컨텐츠를 가려줍니다.
$target : 배경을 넣을 타겟 선택자
$opacity : 배경의 투명도 (기본값 "0.7")
$bgColor : 배경색 (가본값 "#000")
open($this) popupWarp 보이게 합니다
$this : 클릭된 버튼을 넘깁니다. (팝업이 닫치게 되면 넘긴 값으로 포커스가 이동합니다.)
$cookieName : 오늘하루 안보기 쿠키 이름을 넣으면 활성화 됩니다.
close( $cookieName ) popupWarp 숨김니다.
$cookieName : 오늘하루 안보기 쿠키 이름을 넣으면 비활성화 됩니다.
openToday($this , $cookieName) popupWarp 보이게 합니다
$this : 클릭된 버튼을 넘깁니다. (팝업이 닫치게 되면 넘긴 값으로 포커스가 이동합니다.)
$cookieName : 오늘하루 안보기 쿠키 이름을 넣으면 활성화 됩니다.
closeToday($cookieName) popupWarp 숨김니다
$cookieName : 오늘하루 안보기 쿠키 이름을 넣으면 이름을 체크하여 "openToday"열때 하루도안 열리지 않게 합니다.
