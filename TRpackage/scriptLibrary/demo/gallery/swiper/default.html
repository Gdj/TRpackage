Swiper

플리킹 스라이딩 플러그인 Swiper

new Swiper(container, options)

설명
기본 위치값은 CSS 에서 설정해야합니다.
아이템을 감싸고있는 ".swiper-wrapper"과 아이템들".swiper-slide"의 클래스이름은 기본적으로 가지고 있어야합니다.
swiper 스타일 : idangerous.swiper.css
swiper 스크립트 : idangerous.swiper-2.7.5.min.js
Swiper 설명
매개변수 매개변수 설명
container Swiper 컨테이너 선택자를 설정합니다.
options Swiper 여러가지 옵션을 오브젝트 타입으로 줄수 있습니다.
Swiper options 설명
options 속성 options 속성 설명
initialSlide 기본값은 0 이고 첨시작되는 인덱스를 설정한다.
loop 기본 값은 "false" 이고 "true" 설정하면 루프 모드로 설정할수 있습니다.
direction 기본 값은 "horizontal" 이고 "vertical" 설정하면 세로로 형으로 모드를 변경할수 있습니다.
effect 기본 값은 "slide" 이고 "fade", "cube", "overflow"로 변경할수 있습니다.
speed 기본 값은 "300" 이고 숫자형태로 움직이는 시간을 정의할수 있습니다.
spaceBetween 기본 값은 0 이고 숫자형태로 item 간격을 정의할수 있습니다.
slidesPerView 슬라이더 컨테이너에 동시에 표현할수있는 슬라이드 겟수를 숫자 형태로 정의 합니다.
autoplay (1000 = 1초) 로시간을 정의하면 자동으로 슬라이드를 합니다.
pagination HTML 태그에 선택자를 너으면 페이지를 자동으로 생성합니다.
paginationClickable 기본 값은 "false" 이고 "true" 설정하면 페이지를 클릭하면 페이지로 이동합니다.
Swiper methods 설명
Swiper 메서드 Swiper 메서드 설명
.activeIndex 진행중인 인덱스 값을 리턴합다.
loop 모드일때는 복제된 값도 포함되니 커스텀 해서 사용
.previousIndex 이전 슬라이드 인덱스 번호
.slideTo(index, speed, runCallbacks) 인덱스 번호로 슬라이드로 실행
slidePrev(runCallbacks, speed) 이전 슬라이드로 진행합니다.
.slideNext(runCallbacks, speed) 다음 슬라이드로 진행합니다.
.stopAutoplay() autoplay 모드일때 진행을 정지 합니다.
.startAutoplay() autoplay 모드일때 진행을 시작 합니다.
.destroy(deleteInstance,cleanupStyles) Swiper 를초기화 합니다. 기본값(true, false)
.update() Swiper 컨테이너크기, 슬라이더크기, 컨텐츠등을 새로 불러오기.
Swiper event 설명
Swiper 이벤트 Swiper 이벤트 설명
onInit(swiper) swiper 초기화 후 실행
onSlideChangeStart(Swiper) 슬라이드 모션이 시작되는 시점에 호출되는 콜백 합수입니다. 인자값으로 현제"Swiper"를 오브젝트로 받아옵니다.
onSlideChangeEnd(Swiper) 슬라이드 모션이 끝나는 시점에 호출되는 콜백 합수입니다. 인자값으로 현제"Swiper"를 오브젝트로 받아옵니다.
onTouchMove(Swiper, event) 손으로 터치 이동할때 콜백 함수 실행

	$(document).ready(function(){

		// Swiper 세팅
		var mySwiper = new Swiper('.gallery_center',{    
			autoplay : 3000,			
			loop: true, 
			pagination: '.pagination',
			paginationClickable: true,
		
			onSlideChangeEnd: function(swiper){										// 움직임이 끝나면 실행
				$(".pagination li").removeClass("current");
				$(".pagination li").eq(swiper.activeLoopIndex).addClass("current");
			}
		})
	 
		// 좌우버튼
	  $('.prev').on('click', function(e){
		e.preventDefault();					//이벤트 객체의 preventDefault 메소드를 실행하면 기본 동작이 취소된다	
		mySwiper.swipePrev()
		///return false ;					// ie9 이하 버전에서는 event.returnValue를 false로 해야 한다. 
	  })
	  $('.next').on('click', function(e){
		e.preventDefault();					//이벤트 객체의 preventDefault 메소드를 실행하면 기본 동작이 취소된다	
		mySwiper.swipeNext()
		//return false ;					// ie9 이하 버전에서는 event.returnValue를 false로 해야 한다. 

	  })
		
		// 컨텐츠에 마우스 올리면 일시정지
	  $(".gallery_center").hover(function(){
		mySwiper.stopAutoplay(); 
	  } ,
	  function(){
		mySwiper.startAutoplay(); 
	  })

	})
    </script>

		mySwiper.slideNext()
		//return false ;					// ie9 이하 버전에서는 event.returnValue를 false로 해야 한다. 

	  })

		// 첫번째활성화
	  $(".pagination li").eq(_index).addClass("current");				// activeLoopIndex / activeIndex 일반	
	 									

		// 썸네일  컨트롤
	  $(".pagination li").on('click', function(){							
		 _index = $(this).index()+1;
		 //console.log("click : "+ _index)
		 mySwiper.slideTo(_index ,700, function(){});
		 $(".pagination li").removeClass("current");
		 $(this).addClass("current");	
		 return false;
	  })
		

	})
  </script>
