ezMark

ezMark을 이용하여 체크박스, 라디오버튼을 이미지로 변경할수있다.
link : ezMark jQuery Plugin(GitHub)

.ezMark( { option } )

설명
기본 위치값은 CSS 에서 설정해야합니다.
버튼 옵션 정의
image : 아이콘다운로드
js : jquery.ezmark.js 다운로드
button 설명
매개변수 매개변수 설명
checkboxCls 체크박스 기본 스타일
checkedCls 체크박스 활성화 스타일
radioCls 라디오버튼 기본 스타일
selectedCls 라디오버튼 활성화 스타일
