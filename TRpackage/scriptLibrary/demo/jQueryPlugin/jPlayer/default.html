jPlayer

이미지 시컨스를 연속재생 시켜줍니다.
link API : http://jplayer.org/latest/developer-guide/

.jPlayer({ option } )

설명
크로스 브라우징 가능한 동영상 플레이어.
ie7,8 버젼에서는 "swfPath" 옵션 필요
option
option 이름 option 설명
swfPath:"" "video"태그가 지원되지 않은 브라우저를 위해 "jquery.jplayer.swf"파일 경로를 넣는다.
supplied setMedia에 적용될 영상파일 확장자 순서
size:objecj {width:"비디오 width크기", height:"비디오 height크기", cssClass:"비디오 모드" }
nativeVideoControls:objecj 디바이스 기본 플레이어 사용 종류
{ ipad: /ipad/, iphone: /iphone/, android: /android/, blackberry: /blackberry/, iemobile: /iemobile/ }
useStateClassSkin:boolen
ready:function jPlayer가 준비 되면 호출
메서드
메서드 이름 메서드 설명 : $("시컨스컨테이너 선택자").jsMovie("메서드 이름")
$(id).jPlayer( "setMedia", Object: media ) : jQuery 적용될 미디어 타입을 넣음 다양한 확장자를 등록할수록 크로스 브라우징 효과
poster : 영상미리보기 이미지
m4v : mp4 영상파일
ogv : ogv 영상파일
webmv : webmv 영상파일
EVENT
EVENT 이름 EVENT 설명
play 재생이 시작하면 호출 됩니다.
pause 일시정지 하면 호출 됩니다.
stop 재생 정지 되면 호출 됩니다.
ended 마지막 프레임에 도달되면 호출 됩니다.
playing 재생 중일때 호출 됩니다.
loaded 이미지가 모두 로드 되면 호출 됩니다.
verbose, function(e,output) 모든 이벤트 및 텍스트를 포함하여 호출 됩니다.
