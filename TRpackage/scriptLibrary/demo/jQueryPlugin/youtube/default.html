YouTube API

YouTube API를 이용한 컨트롤

YT.Player( selector ,{ option } )

설명
기본 위치값은 CSS 에서 설정해야합니다.
버튼 옵션 정의
option
매개변수 매개변수 설명
width 동영상 플레이어의 너비입니다. 기본값은 640입니다.
height 동영상 플레이어의 높이입니다. 기본값은 390입니다.
videoId 플레이어에서 로드할 동영상을 식별하는 YouTube 동영상 ID입니다
events 개체의 속성은 API가 실행하는 이벤트 및 해당 이벤트 발생 시 API가 호출하는 함수(이벤트 리스너)를 식별합니다. 예를 들어 생성자는 onReady 이벤트 실행 시 onPlayerReady 함수가 실행되고 onStateChange 이벤트 실행 시 onPlayerStateChange 함수가 실행됨을 나타냅니다.
메서드
메서드 메서드 설명
.playVideo() 현재 신호를 준 동영상과 로드한 동영상을 재생합니다. 이 함수를 실행한 후 플레이어의 마지막 상태는 playing(1)입니다.
pauseVideo() 현재 재생 중인 동영상을 일시중지합니다. 함수가 호출될 때 플레이어 상태가 변경되지 않는 ended(0) 상태에 있지 않는 한 이 함수를 실행한 후 플레이어의 마지막 상태는 paused(2)입니다.
stopVideo() 현재 동영상의 로드를 중지하고 취소합니다. 이 함수는 사용자가 플레이어에서 추가 동영상을 보지 않을 것을 알고 있는 드문 경우에 사용해야 합니다. 동영상을 일시중지하려는 경우에는 pauseVideo 함수를 호출해야 합니다. 플레이어가 재생 중인 동영상을 변경하려는 경우 stopVideo를 먼저 호출하지 않고 대기열 함수 중 하나를 호출할 수 있습니다.
seekTo(seconds:Number, allowSeekAhead:Boolean) 동영상에서 지정한 시간을 찾습니다. 함수가 호출될 때 플레이어가 일시중지되면 일시중지로 유지됩니다.
clearVideo() 동영상 표시를 지웁니다. 이 함수는 stopVideo() 호출 후 잔여 동영상을 지우려는 경우에 유용합니다.
cueVideoById(videoId:String, startSeconds:Number, suggestedQuality:String) 동영상 변경
videoId : 동영상 ID
startSeconds : 시작 초
suggestedQuality : 동영상 쿼릴티 (small, medium, large, hd720, hd1080, highres, default) default 권장
mute() 플레이어를 음소거합니다.
unMute() 플레이어 음소거를 해제합니다.
isMuted() 플레이어가 음소거 상태인 경우 true를 반환하고 음소가 상태가 아닌 경우 false를 반환합니다.
setVolume(volume:Number) 볼륨을 설정합니다. 0에서 100 사이의 정수를 허용합니다.
getVolume() 플레이어의 현재 볼륨을 0에서 100 사이의 정수로 반환합니다.
setSize(width:Number, height:Number) 플레이어가 포함된 "iframe"의 크기를 픽셀 단위로 설정합니다.
setPlaybackRate(suggestedRate:Number) 이 함수는 현재 동영상에 대한 추천 재생 속도를 설정합니다. 재생 속도가 변경되면 이미 신호를 주었거나 재생 중인 동영상에 대해서만 변경됩니다.
getPlaybackRate() 이 함수는 현재 재생 중인 동영상의 재생 속도를 가져옵니다. 기본 재생 속도는 1이며 동영상이 일반적인 속도로 재생되고 있음을 나타냅니다. 재생 속도에는 0.25, 0.5, 1, 1.5 및 2와 같은 값이 포함될 수 있습니다.
getDuration() 현재 재생 중인 동영상의 재생 시간을 초 단위로 반환합니다
getVideoUrl() 현재 재생 중인 동영상의 재생 시간을 초 단위로 반환합니다
event
이벤트 이벤트 설명
onReady 동영상 플레이어가 준비되면 호출 될함수를 설정
onStateChange($event) 이 이벤트는 플레이어의 상태가 변경될 때마다 실행됩니다
$event.data 값 아래 참고.
  • -1 //(시작되지 않음)
  • 0 == YT.PlayerState.ENDED //(종료됨)
  • 1 == YT.PlayerState.PLAYING //(재생 중)
  • 2 == YT.PlayerState.PAUSED //(일시중지됨)
  • 3 == YT.PlayerState.BUFFERING //(버퍼링 중)
  • 5 == YT.PlayerState.CUED //(재생 준비되면)
onPlaybackQualityChange($suggestedQuality:String) 동영상 재생 품질이 변경될 때마다 실행됩니다.
  • default : 좋은방법
  • small
  • medium
  • large
  • hd720
  • hd1080
  • highres
onPlaybackRateChange 이벤트는 동영상 재생 속도가 변경될 때마다 실행됩니다
onError 플레이어에서 오류가 발생하면 실행됩니다.
event

      var player;
      function onYouTubeIframeAPIReady() { 	
				player = new YT.Player('player', {
					height: '390',
					width: '640',
					videoId: 'M7lc1UVf-VE',					
					events: {
					'onReady': playVideo					
				}
        });
      }

  
	  /* 재생 */
      function playVideo() {
        player.playVideo();
      }

	  /* 재생정지 */
      function stopVideo() {
		player.seekTo(0, true );
        player.stopVideo();		
        player.clearVideo(); //잔여영상지우기
      }

	  /* 재생 일시정지*/
	  function pauseVideo() {
        player.pauseVideo();
      }

	  /* seekTo */
	  function seekTo($seconds, $allowSeekAhead ) {
        player.seekTo($seconds, $allowSeekAhead );
      }


	  /* 음소거 해지 => player.isMuted() : 플레이어가 음소거 상태인 경우 true를 반환하고 음소가 상태가 아닌 경우 false를 반환합니다.*/	
      function unMute() {
        player.unMute();
      }

	  /* 음소거 */	
      function mute() {
        player.mute();
      }	
	  
  	  /* 볼륨 조절 : 볼륨을 설정합니다. 0에서 100 사이의 정수를 허용합니다. */
      function setVolume($volume) {
        player.setVolume($volume);
		// player.getVolume()  볼륨 반환
      }


	  /* 재생속도 변경 ( x1=1 , x22=2.5 )*/
      function setPlaybackRate($suggestedRate) {
        player.setPlaybackRate($suggestedRate);
		// player.getPlaybackRate():Number 이 함수는 현재 재생 중인 동영상의 재생 속도를 가져옵니다.  
      }
