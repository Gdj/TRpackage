media query

설명
컨테이너 쿼리 사용 방법
클래스(.containerQuery)에 컨테이너이름(div-container)을 부여하여 사용
@container div-container (width <= XXXpx) : 데스크탑 기준 : XXXpx 까지
layer Popup 설명
메서드 메서드 설명
container-type: 컨테이너 타입 [normal(컨테이너 쿼리 비활성화), size(가로세로 모두반응), inline-size(기로반응), scroll-state(스크롤에따라), inherit(부모 쿼리에 따라), inital(생성시쿼리에 따라)]
contain: 웹 페이지의 렌더링 성능을 최적화하기 위해 사용되는 CSS 속성
[none(기본값. 컨테인먼트(containment)를 적용하지 않음),
strict(모든 종류의 컨테인먼트를 적용함),
size(요소의 크기가 외부 콘텐츠의 영향을 받지 않도록 함),
layout(요소의 내부 레이아웃이 외부 레이아웃에 영향을 주지 않도록 함), style(요소의 콘텐츠 내부에 있는 CSS 속성(카운터, 폰트, quotes 등)이 외부 콘텐츠에 영향을 주지 않도록함),
paint(요소의 내부 콘텐츠가 외부 콘텐츠 위로 넘쳐 흐르거나(overflow) 다른 요소의 렌더링에 영향을 주지 않도록 함) ]
container-name 컨테이너 이름 지정
이름이 없을때 부모컨테이너에 아래속성이 있는 더 가까운 컨테이너에 반응함.
container-type: inline-size;
container-type: size;
contain: layout;
contain: style;
