layer Popup 플러그인 bPopup API
jquery.bpopup.min.js : download

.bPopup( options )

설명
기본 위치값은 CSS 에서 설정해야합니다.
options 설명
options 이름 options 설명
zIndex zIndex 조절
modalClose 모달 팝업 사용여부 (기본값 : true)
modal 모달 팝업 사용여부 (기본값 : true)
modalClose 모달 팝업 overlayer 클릭시 닫침여부 (기본값 : true)
opacity 투명도 값 (기본 : 0.7)
position : [x,y] 팝업 위치 (기본값 : ["auto", "auto"] )
layer Popup Event 설명
메서드 메서드 설명
callback:function(){} 팝업이 로드 된후 실행
onOpen:function(){} 팝업이 열리기전 실행
onClose:function(){} 팝업이 닫치기전 실행
layer Popup 설명
메서드 메서드 설명
close() 팝업 닫기
