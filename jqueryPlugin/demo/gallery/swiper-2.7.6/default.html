Swiper 2.7.6

플리킹 스라이딩 플러그인 Swiper

new Swiper(container, options)

설명
기본 위치값은 CSS 에서 설정해야합니다.
아이템을 감싸고있는 ".swiper-wrapper"과 아이템들".swiper-slide"의 클래스이름은 기본적으로 가지고 있어야합니다.
swiper 스타일 : idangerous.swiper.css
swiper 스크립트 : idangerous.swiper-2.7.5.min.js
Swiper 설명
매개변수 매개변수 설명
container Swiper 컨테이너 선택자를 설정합니다.
options Swiper 여러가지 옵션을 오브젝트 타입으로 줄수 있습니다.
Swiper options 설명
options 속성 options 속성 설명
loop 기본 값은 "false" 이고 "true" 설정하면 루프 모드로 설정할수 있습니다.
mode 기본 값은 "horizontal" 이고 "vertical" 설정하면 세로로 형으로 모드를 변경할수 있습니다.
speed 기본 값은 "300" 이고 숫자형태로 움직이는 시간을 정의할수 있습니다.
slidesPerView 슬라이더 컨테이너에 동시에 표현할수있는 슬라이드 겟수를 숫자 형태로 정의 합니다.
autoplay (1000 = 1초) 로시간을 정의하면 자동으로 슬라이드를 합니다.
pagination HTML 태그에 선택자를 너으면 페이지를 자동으로 생성합니다.
paginationClickable 기본 값은 "false" 이고 "true" 설정하면 페이지를 클릭하면 페이지로 이동합니다.
Swiper methods 설명
Swiper 메서드 Swiper 메서드 설명
.swipeTo(index, speed, runCallbacks) 인덱스 번호로 슬라이드로 실행
.swipePrev() 이전 슬라이드로 진행합니다.
.swipeNext() 다음 슬라이드로 진행합니다.
.activeIndex 진행중인 인덱스 값을 리턴합다..
.activeLoopIndex loop 모드일때 진행중인 인덱스 값을 리턴합다.
.stopAutoplay() autoplay 모드일때 진행을 정지 합니다.
.startAutoplay() autoplay 모드일때 진행을 시작 합니다.
Swiper event 설명
Swiper 이벤트 Swiper 이벤트 설명
onSlideTouch(Swiper) 터치 슬라이딩 후 바로 호출되는 콜백 합수입니다.
.clickedSlide , .clikedSlideIndex 값을 사용할수있습니다.
onSlideChangeStart(Swiper) 슬라이드 모션이 시작되는 시점에 호출되는 콜백 합수입니다. 인자값으로 현제"Swiper"를 오브젝트로 받아옵니다.
onSlideChangeEnd(Swiper) 슬라이드 모션이 끝나는 시점에 호출되는 콜백 합수입니다. 인자값으로 현제"Swiper"를 오브젝트로 받아옵니다.
