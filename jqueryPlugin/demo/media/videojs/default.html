videojs( id이름 )

설명
IE11(Win8.1 이상), Edge, 크롬, FF 지원
네이티브 메서드이름 함수형태로 지원.
메서드
메서드 이름 메서드 설명
videojs("ID명"); video 브라우져마다 도일한 스킨적용
poster : 영상미리보기 이미지
m4v : mp4 영상파일
ogv : ogv 영상파일
webmv : webmv 영상파일
methods
methods 이름 methods 설명
.canPlayType() 지정된 미디어 유형을 재생할 수 있는지 여부를 결정합니다.
.fastSeek() 주어진 시간을 직접 찾습니다.
.load() 미디어 요소를 재설정하고 미디어 리소스를 다시 시작합니다.
.pause() 미디어 재생을 일시 중지합니다.
.play() 미디어 재생을 시작합니다.
.currentTime(초) 해당 위치로 플래이 해드옴김
.paused() 재생중인지 부울린값 및 설정

  var myVideo = videojs('example_video_1');
  
  function video_play(){
    myVideo.play();
    } 
    function video_pause(){
      myVideo.pause();
    }
    function video_stop(){
      myVideo.currentTime(0);
      myVideo.pause();
      myVideo.load();
    }
    console.log(myVideo)
    function video_playPause() {
        if (myVideo.paused())
            myVideo.play();
        else
            myVideo.pause();
    }
