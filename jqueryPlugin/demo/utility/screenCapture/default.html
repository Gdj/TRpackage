html2canvas
옵션 이름 옵션 설명
async (true) 요소를 비동기 적으로 구문 분석하고 렌더링할지 여부
canvas (null)그리기의 기초로 사용할 기존 캔버스 요소.
onclone (null)문서가 렌더링을 위해 복제되었을 때 호출되는 콜백 함수는 원본 소스 문서에 영향을 미치지 않고 렌더링 될 내용을 수정하는 데 사용할 수 있습니다.
removeContainer (true)Whether to cleanup the cloned DOM elements html2canvas creates temporarily
width, height (Element width, height)The width of the canvas
x, y (Element x-offset, y-offset) 자르기 캔버스 x, y 좌표
scrollX, scrolly (Element scrollX, scrollY) 요소를 렌더링 할 때 사용되는 x, y 스크롤 위치 (예 : Element가 position : fixed를 사용하는 경우)
windowWidth, windowHeight (Window.innerWidth, Window.innerHeight) 요소를 렌더링 할 때 사용할 창 너비, 높이로 미디어 쿼리와 같은 것에 영향을 줄 수 있습니다.
