stickyfill

stickyfill 이용하여 position sticky 적용
link : stickyfill jQuery Plugin(GitHub)

Stickyfill.add($('.class'));

설명
기본 위치값은 CSS 에서 설정해야합니다.
top은 뷰포트의 상단 가장자리에 상대적인 고정 위치를 지정합니다. 음수 값도 허용합니다.
양수 또는 음수 margin-bottom을 지정하여 스티키의 하단 한계를 위 또는 아래로 밀 수 있습니다.
스크롤시 부모 컨테이너 크기에 고정됩니다.
CSS 옵션
속성 속성 설명
position:sticky 커스텀 네비 선택자
top sticky 시 컨테이너 상단으로 부터의 거리
margin-bottom sticky 시 컨테이너 하단 거리
masonry 메서드
메서드 메서드 설명
.addOne(element) 목록의 첫번째 요소 사용. (사용요소 반ㅁ환)
.removeOne(element) 목록의 첫번째 요소 스티키 제거.
.add(elementList) 목록의 요소 사용. (사용요소 배열로 반환)
.remove(elementList) 목록의 요소 사용 스티키 제거
.refreshAll() 모든 기존 스티커를 새로 고치고 매개변수와 위치를 업데이트합니다.
.forceSticky() 브라우저가 position:sticky를 기본적으로 지원하더라도 폴리필을 강제로 활성화합니다.
.stickies 기존 고정 인스턴스의 배열입니다.
