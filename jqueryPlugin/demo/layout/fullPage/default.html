fullPage.js

fullPage 이용하여 원페이지 사이트 .
doc link : fullPage jQuery Plugin(GitHub)

.fullPage( { option } )

설명
기본 위치값은 CSS 에서 설정해야합니다.
사용 리소스 : [jquery.fullPage.css, jquery-1.12.1.min.js, jquery.fullPage.js]
버튼 옵션 정의
masonry 옵션
매개변수 매개변수 설명
menu 커스텀 네비 선택자
anchors 배열형대로 앵커 네임 생성
sectionsColor 배열형대로 배경색 지정
navigation 기본값은 false, 이며 true 설정하면 현제 위치를 도트로 표시한다.
navigationPosition 기본값"none", 이며 "left" or "right"로정열 한다.
normalScrollElements 스크롤 이벤트를 제외 하고 싶은 선택자 "#element1, .element2"
scrollBar 기본값은 false, 이며 true 이면 바디 스크롤을 생성한다.
keyboardScrolling 기본값은 true 이며 false 로설정하면 키보드 접근을 막는다.
masonry 메서드
메서드 메서드 설명
.moveSectionUp() 슬라이드 위로 한칸 가기.
.moveSectionDown() 슬라이드 아래로로 한칸 가기.
.moveTo(section, slide) 섹션 위치와, 섹션의 슬라이드 위치로 이동.
.silentMoveTo(section, slide) 섹션 위치와, 섹션의 슬라이드 위치로 모션없이 이동
.moveSlideLeft(), .moveSlideRight() 색션의 슬라이드 좌우로 이동
.setAutoScrolling(true:boolean) 스크롤 한번에 페이지 넘김 여부
.setFitToSection(true:boolean) 섹션 슬라이드 스넵걸기 세팅.
.fitToSection() 현제 섹션 스넵 위치이동.
.setAllowScrolling(boolean, [directions]) directions : all, up, down, left, right
현제 스크롤 이벤트 제한 하기.
.setLockAnchors(true:boolean) 앵커가 URL에 영향을 미칠지 여부를 결정하는 lockAnchors 옵션의 값을 설정합니다.
masonry 이벤트
이벤트 이벤트 설명
onLeave function(index, nextIndex, direction)
현색션을 떠날때 호출
afterLoad function(anchorLink, index)
스크롤이 끝난후 섹션이 로드되면 호출
afterRender 페이지의 구조가 생성 된 직후 호출, 다른 플러그인을 초기화 할때 호출
afterResize 브라우저 창 크기를 조정 한 후에 호출
