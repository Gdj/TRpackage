ScrollMagic 2.0.7

스크롤에 반응하는 이벤트로 속성변화.
doc link : ScrollMagic DOC
demo link : ScrollMagic DEMO
사용 리소스 : [ScrollMagic.min.js, debug.addIndicators.js]

var controller = new ScrollMagic.Controller({ ... });
var scene = new ScrollMagic.Scene({ ... }).setPin("").addTo(controller);

설명
jQuery 3.3.1

# Controller

Controller 설명
Controller 옵션 options 설명
globalSceneOptions({triggerHook}) 브라우져 위상값(트리거위치)
(문자, 분율) : (['onLeave', onCenter, onEnter], [0, .5, 1])
globalSceneOptions({duration}) 지속될크기 px : Number
지속될크기 "70%" : String

# Scene

Scene 설명
Scene 옵션 options 설명
triggerElement : 시작위치 선택자 : string
duration : 지속될크기 px : Number
지속될크기 "70%" : String
offset : 이밴트 영역을 미루고 당김 px : Number
triggerHook : 브라우져 위상값(트리거위치)
(문자, 분율) : (['onLeave', onCenter, onEnter], [0, .5, 1])
reverse: 스크롤 up, donw 반응여부 : true
스크롤 donw 한번만 실행됨 : false
enabled: 적용 여부 : true
tween: 프러그인 모션 여부 : true
pin: 타겟 적용여부 : true

# method

method 설명
method method 설명
setPin("", {})
  • - 시작위치 선택자 : string ,
  • - {pushFollowers:true} :
    true end 영역까지 밀어냄. 핀이 유지되는 동안 다음 요소가 아래로 "밀려지는" 경우,
    false 컨텐츠 크기로 정의됨. 고정된 요소가 해당 요소를 지나서 스크롤되는 경우
setClassToggle("$selecter","$class") $selecter : 선택자
$class : 적용할 클래스 이름
addIndicators({name}) 가이드에 들어갈 문자
addTo(controller) 컨트롤러를 반영 시킴

# event

event 설명
event event 설명 : scene.on("update", function ($e) { console.log($e); });
update 장면이 업데이트될 때마다 발생합니다.
  • $e.type : 이벤트 이름
  • $e.target : 이벤트를 트리거한 Scene 객체
  • $e.startPos : 장면의 시작 위치(컨테이너 기준)
  • $e.endPos : 장면의 끝 위치(컨테이너 기준)
  • $e.scrollPos : 컨테이너의 현재 스크롤 위치
start, end 장면이 시작점 도달, 장면 끝점 도달
  • $e.type : 이벤트 이름
  • $e.target : 이벤트를 트리거한 Scene 객체
  • $e.progress : 장면의 현재 진행 상황을 반영합니다.
  • $e.state : 현재 현장 상황 start(이전:"BEFORE", 들어옴:"DURING"), end(들어옴"DURING", 나감:"AFTER)
  • $e.scrollDirection : 스크롤하는 방향을 나타 냅니다 "PAUSED", 스크롤 다운"FORWARD", 스크롤 업"REVERSE"
enter, leave 방향 성없이 장면이 들어옴 도달, 장면 벗어남
  • $e.type : 이벤트 이름
  • $e.target : 이벤트를 트리거한 Scene 객체
  • $e.progress : 장면의 현재 진행 상황을 반영합니다.
  • $e.state : 현재 현장 상황 enter(들어옴:"DURING"), (이전:"BEFORE", 나감:"AFTER)
  • $e.scrollDirection : 스크롤하는 방향을 나타 냅니다 "PAUSED", 스크롤 다운"FORWARD", 스크롤 업"REVERSE"
addIndicators({name}) 가이드에 들어갈 문자
addTo(controller) 컨트롤러를 반영 시킴

			 }
		 });
 
		 // get all slides
		 var slides = document.querySelectorAll(".js-item");
 
		 // create scene for every slide
		 for (var i=0; i<slides.length; i++) {
			 new ScrollMagic.Scene({ triggerElement: slides[i] })

	 });
	 </script>
